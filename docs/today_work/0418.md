### 4. 小技能现在是共享抛物线跳跃，不再是 blink 式位移

这一步主要收了 4 件事：

- `Stage01SampleContentBuilder.cs` 里的 `AddRepositionEffect(...)` 现在会同步写入 `forcedMovementDurationSeconds`
- `BattleSkillSystem.ApplyDashReposition(...)` 当前会先读取重定位 effect 的 `durationSeconds`，未填写时再退回 `forcedMovementDurationSeconds`
- `Tidal Pounce.asset` 当前位移参数改成了：
  - `durationSeconds = 0.28`
  - `forcedMovementDurationSeconds = 0.28`
  - `forcedMovementPeakHeight = 0.85`
- `BattleView` 现在会把“带明显离地高度的 Dash/Reposition”视为跳跃，而不是瞬移或平面冲锋

当前表现规则已经拆成三类：

- 真瞬移：
  - 仍然可以走 blink 残影
- 平面冲锋：
  - 仍然可以走地面 dash 拖尾
- 带抛物线高度的位移：
  - 不再播放 blink 残影
  - 不再套地面 dash charge 拖尾
  - 主要复用已有的离地、落地、影子缩放和落地 pulse 反馈

## 这轮最终定稿的英雄方案

当前应以 `docs/heroes/assassin/tidefin.md` 为腐蚀虫的直接规格来源。

核心规则如下：

- 英雄名：
  - 中文：`腐蚀虫`

  - heroId：`assassin_002_tidefin`
- 普攻：
  - 近战普攻
  - 命中后使目标 `攻击力 -5%`
  - 持续 `3 秒`
  - 可叠加，最多 `5` 层

  - 跳向 `4` 范围内累计造成伤害最高的敌人

  - 持续 `5 秒`
- 大招：
  - 对 `4` 范围内所有敌人施加减益
  - `攻击力 -20%`
  - `防御力 -20%`
  - 持续 `5 秒`

- 如果“累计输出”和“当前生命”都并列，则取更近的那个目标。

补这条不是为了改设计，而是为了避免运行时在完全并列时出现不稳定选目标。
