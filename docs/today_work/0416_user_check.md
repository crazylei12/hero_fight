# 0416 用户观察清单

最后更新：2026-04-16

## 文档用途

这份文档不是给代码审查 AI 的。

它只用于后续你自己打开游戏时，快速对照观察这轮“远程受击后撤”实现到底表现得对不对。

重点不是看代码是否优雅，而是看画面和行为是否符合预期。

## 建议观察顺序

建议你分两种场景观察：

1. 先看更纯一点的普攻场景
2. 再看带技能的正式战斗场景

如果当前工程里都能打开，优先建议：

1. `Assets/Scenes/BattleBasicAttackOnly.unity`
2. `Assets/Scenes/Battle.unity`

原因：

- 前者更容易看清 retreat 到底是不是来自普攻 threat
- 后者更容易看出技能、DOT、团战混在一起时是否会出问题

## 你要重点看的现象

### 1. 射手 / 法师被打后，是否会短暂后撤

你要看的是：

- 敌人打到射手或法师后
- 他们是否会往远离敌人的方向退一点
- 退完后是否还能继续恢复输出

理想现象：

- 有明显“被压一下就拉开一点”的感觉
- 但不是一直跑

异常现象：

- 被打了完全不退
- 一退就停不下来
- 一直贴着边走，不再回头输出

### 2. 近战英雄是否保持原本行为

你要看的是：

- 战士、坦克、刺客被打时
- 是否仍然主要以前压、接敌、近身输出为主

理想现象：

- 近战整体行为和这次改动前差不多

异常现象：

- 近战也开始频繁后撤
- 前排莫名其妙不愿意接敌

### 3. 未受击时，远程是否还能正常站位输出

你要看的是：

- 射手 / 法师在没有明显被打时
- 是否仍按原本节奏索敌、前压、输出

理想现象：

- 没受压时和旧行为接近

异常现象：

- 还没被打就开始自己后退
- 远程整体变得很怂，明显不愿意进攻

### 4. 是否出现边缘距离抖动

这是最重要的观察点之一。

你要看的是：

- 远程和敌人接近攻击距离边缘时
- 会不会前进一步、后退一步、再前进一步
- 形成肉眼可见的来回抽搐

理想现象：

- 行为有节奏，偶尔拉扯，但不抖

异常现象：

- 原地鬼畜横跳
- 距离边缘疯狂来回拉扯

### 5. retreat 会不会让远程几乎打不出输出

你要看的是：

- 射手 / 法师在团战里是不是老在退
- 退的频率是否高到攻击次数明显不足

理想现象：

- retreat 只是补充行为，不会夺走主要输出节奏

异常现象：

- 大部分时间都在跑
- 明显伤害掉很多
- 团战里几乎不出手

### 6. Support 会不会保守过头

这条要特别看 `Sunpriest`。

你要看的是：

- 辅助被打后会不会后撤
- 后撤会不会让她很难继续治疗友军

理想现象：

- 被贴脸时会稍微自保
- 但整体仍能稳定站在中后排做治疗

异常现象：

- 一受压就一直跑
- 经常因为 retreat 断掉治疗节奏
- 队友残血时她还在后面绕圈

### 7. DOT / 持续伤害会不会引发异常逃跑

你要看的是：

- 英雄站在持续伤害区域里
- 或身上挂了持续伤害时
- 是否会因为 threat 被不断刷新而变得异常保守

理想现象：

- 有一定自保倾向，但还保持正常作战

异常现象：

- 被 DOT 烧到后一直退，不再正常回战
- 某些英雄像被“吓坏了一样”持续逃跑

### 8. 战斗日志和目标切换是否变得异常吵

如果你会顺手看 HUD / 日志，也建议一起看。

你要看的是：

- 目标切换是不是突然多很多
- retreat 过程中是否不断刷新的目标变化很夸张

理想现象：

- 日志量增加一点可以接受

异常现象：

- `TargetChanged` 明显刷屏
- 一个单位短时间内反复改目标

## 最后你需要给我的反馈

你后面看完以后，最有价值的反馈不是“感觉还行”，而是下面这种：

- 哪个职业表现最好
- 哪个职业表现最怪
- 有没有明显抖动
- 有没有明显后撤过头
- support 是否被误伤
- DOT / 技能混战时有没有很奇怪的 retreat

如果你时间不多，至少优先回答这 5 个问题：

1. 射手被打后有没有出现你想要的“走 A 拉扯感”？
2. 会不会明显抖动？
3. 会不会退得太多导致不输出？
4. 近战有没有被误伤？
5. Sunpriest 会不会因为 retreat 影响治疗？
