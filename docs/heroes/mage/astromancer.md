# Astromancer

## 基础信息

- 英文名：Astromancer
- 中文名：星术师
- 英雄 ID：`mage_005_astromancer`
- 英雄分类：法师
- 标签：远程、延迟命中、预判轰炸、区域惩罚、范围伤害
- 核心定位：中后排预判型法师，所有主动技能都有短暂落点延迟；单次命中不如火法稳定，但命中被减速、被控制或抱团停留的敌人时收益更高
- 默认 AI 模板：`mage_default`
- 当前定位补充：作为 `mage_004_lightningmage` 之后的第五个法师扩展原型，不替代 `Firemage` 的直接范围爆发，不替代 `Frostmage` 的减速控场，不替代 `Sandemperor` 的阵地部署，也不替代 `Lightningmage` 的高频小控；主要验证“落点快照 + 延迟命中 + 区域停留惩罚”这条预判轰炸法师方向

## 当前数值倾向

- 星术师的技能应允许高收益，但必须通过“延迟后仍命中”来兑现，不应做成稳定必中瞬间爆炸
- 星术师自身不提供强减速、硬控或聚怪能力；她的上限应主要来自队友提供的减速、眩晕、击飞、拉拽或阵型压缩
- 对快速移动、频繁位移或分散站位的敌人，星术师应明显掉强；对移动慢、被控制或抱团停留的敌人，她应能打出高额范围伤害
- 大招总伤害可以高于普通持续区域技能的单跳威胁，但因为每次星陨都有预警与延迟，实战满额吃满应比较困难

## 普攻类型

- 类型：远程投射物普攻
- 释放目标：当前索敌目标；默认优先攻击最近的可攻击敌人，沿用现有法师和远程投射物英雄的共享索敌与受压后撤基线
- 效果：
  - 发射一枚星光弹，对当前目标造成基于攻击力 `1.0` 倍率的单体伤害
  - 普攻只承担稳定补伤职责，不附带减速、控制或额外标记，避免把星术师变成自带命中保障的全能法师
- 检测条件：
  - 当前存在合法敌方目标
  - 目标进入攻击范围
  - 普攻间隔转好
- 当前关键参数：
  - 攻击距离：`8.9`
  - 普攻间隔：`1.25 秒`
  - 投射物速度：`14`
  - 普攻伤害倍率：基于攻击力 `1.0`

## 小技能：Falling Star

- 技能类型：延迟落点范围伤害
- 释放目标：对当前攻击目标的当前位置做落点快照；若当前目标失效或超出施法距离，则可回退到施法距离内敌方较密集区域的中心点
- 效果：
  - 在目标点生成一个短暂预警落点
  - 经过短暂延迟后，一颗坠星落下并对范围内敌方英雄造成伤害
  - 落点在施法时或预警开始时快照，坠星不会追踪目标后续移动
  - 若目标在延迟期间快速离开中心区域，只会吃到较低边缘伤害或完全躲开
  - 中心命中造成基于攻击力 `1.45` 倍率的范围伤害
  - 外圈擦边命中造成基于攻击力 `0.65` 倍率的范围伤害
  - 当前语义重点是“预判落点打中后很赚”，不是“按下技能就稳定炸到当前目标”
- 检测条件：
  - 小技能 CD 转好
  - 当前存在合法敌方目标，或施法范围内能找到合法敌方区域锚点
  - 预计落点范围内至少存在 `1` 名敌方英雄
  - 当前没有被硬控到无法施法
- 当前关键参数：
  - 施法距离：`9.5`
  - 预警 / 坠落延迟：`0.75 秒`
  - 中心半径：`1.6`
  - 外圈半径：`3.0`
  - 中心伤害倍率：基于攻击力 `1.45`
  - 外圈伤害倍率：基于攻击力 `0.65`
  - 冷却时间：`6.5 秒`

## 大招：Meteor Choir

- 技能类型：持续区域轰炸 / 多段延迟范围伤害
- 释放目标：优先选择施法距离内敌方最密集区域；若区域内已有减速、硬控、拉拽后聚集或其他限制移动条件，则显著提高释放优先级
- 效果：
  - 在目标区域建立一片固定的群星轰炸区
  - 大招区域固定在释放瞬间选定的位置，不跟随星术师或初始目标移动
  - 持续期间连续安排数颗小星陨落下
  - 每颗小星陨都会先记录一个落点快照并生成短暂预警，随后才结算命中
  - 每次落点优先从仍停留在大招区域内的敌方英雄当前位置或小范围密集点中选择；落点确定后不再追踪移动
  - 单颗星陨造成基于攻击力 `1.25` 倍率的范围伤害
  - 同一敌人如果持续停留在大招区域内，可能连续吃到多段伤害；如果及时离开区域或不断变向，命中段数会明显下降
  - 大招核心收益是持续逼迫敌人离开危险区域，并惩罚被减速、被控制或被聚集的目标，不是瞬间一发清场
- 检测条件：
  - 本局尚未释放过大招
  - 优先要求施法距离 `10.5` 内能找到半径 `7.2` 范围内至少存在 `3` 名敌人的区域锚点
  - 若开战 `30 秒` 后仍未满足，则阈值降低为至少 `2` 名敌人
  - 若开战 `45 秒` 后仍未满足，则阈值降低为至少 `1` 名敌人
- 当前关键参数：
  - 施法距离：`10.5`
  - 大招区域半径：`7.2`
  - 单颗星陨命中半径：`2.4`
  - 区域持续时间：`5 秒`
  - 星陨数量：`7`
  - 星陨安排间隔：`0.7 秒`
  - 单颗星陨预警 / 坠落延迟：`0.55 秒`
  - 单颗伤害倍率：基于攻击力 `1.25`
- 当前关键约束：
  - 主动释放型大招本局仅能释放 `1 次`

## 落地约束

- 星术师当前定位为“第五个法师扩展原型”，不替代 `Firemage` 的范围爆发，也不替代 `Frostmage` 的减速压制、`Sandemperor` 的部署阵地或 `Lightningmage` 的感电短控
- `Falling Star` 和 `Meteor Choir` 都应优先复用共享的“延迟落点范围命中”能力表达：记录目标点快照、等待预警延迟、再按范围收集命中目标
- 落点快照不应在延迟期间继续锁定或追踪原目标；如果目标移动离开，技能应允许落空或只命中边缘
- 如果实现层暂时不支持内圈 / 外圈伤害衰减，第一版可以先用单一命中半径近似，但文档语义仍保留“中心收益更高、边缘收益更低”的方向，后续应补成共享配置能力而不是星术师私有逻辑
- `Meteor Choir` 应优先复用持续区域 + 周期安排子落点 + 延迟范围命中的组合结构，不要写成火法 `Meteor Fall` 的即时持续 tick 换皮
- 大招每颗星陨的落点、预警、命中目标数量和造成伤害都应进入统一事件与日志，方便复查命中不稳定是否符合设计
- 星术师的英雄级 AI 当前应继续沿用法师中后排站位基线，不额外引入专属逃跑、贴身拉扯或自带控场行为

## 设计边界

- 星术师的核心职责是延迟轰炸和区域停留惩罚，不承担稳定瞬间爆发主炮职责
- 不应把星术师做成火法换皮；她的技能必须保留明显预警和可躲避窗口
- 不应给星术师额外塞强减速、眩晕、击飞或拉拽来保证自己命中；这部分配合空间应留给队友
- 不应把 `Meteor Choir` 做成全场必中落雷或无预警多段核爆；敌人及时离开区域应能显著降低吃到的段数
- 不应把星术师做成高机动、自保极强或前排站场法师；她仍应依赖中后排站位和队友保护
- 第一阶段下，星术师优先验证“远程投射物普攻 + 延迟快照落点小技能 + 一局一次固定区域多段预警星陨”这套预判轰炸法师模板
