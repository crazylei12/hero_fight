# Frostmage

## 基础信息

- 英文名：Frostmage
- 中文名：冰霜法师
- 英雄 ID：`mage_002_frostmage`
- 英雄分类：法师
- 标签：远程、控制、范围减速
- 核心定位：中后排控场法师，优先朝敌方密集区域铺设减速区域，压低敌方推进、追击和拉扯效率
- 默认 AI 模板：`mage_default`
- 当前定位补充：作为 `mage_001_firemage` 之后的第二个法师原型，不替代火法的范围爆发模板，主要验证“范围减速压制型法师”

## 普攻类型

- 类型：远程投射物普攻
- 释放目标：当前索敌目标；默认优先攻击最近的可攻击敌人，技能仍优先寻找敌方较密集区域
- 效果：造成基于攻击力 `1.0` 倍率的单体伤害
- 检测条件：
  - 当前存在合法敌方目标
  - 目标进入攻击范围
  - 普攻间隔转好
- 当前关键参数：
  - 攻击距离：`5.8`
  - 普攻间隔：`1.30 秒`
  - 投射物速度：`14`

## 小技能：Frost Burst

- 技能类型：范围伤害 + 移速减益 Debuff
- 释放目标：朝施法范围内敌方最密集区域释放
- 效果：
  - 在目标点引发一次冰霜爆裂
  - 对区域内敌人造成基于攻击力 `0.9` 倍率的范围伤害
  - 同时为命中的敌人附加 `MoveSpeedModifier`
  - 减速幅度为 `-30%`
  - 减速持续 `2.5 秒`
  - 当前参数下更偏向“打断敌方前压节奏并拖慢切入”，而不是依靠单次爆发直接压低血线
- 检测条件：
  - 小技能 CD 转好
  - 施法距离 `6.2` 内能找到合法敌方区域锚点
  - 当前至少能命中 `1` 个敌人
- 当前关键参数：
  - 施法距离：`6.2`
  - 区域半径：`2`
  - 伤害倍率：基于攻击力 `0.9`
  - 减速幅度：`-30%`
  - 减速持续时间：`2.5 秒`
  - 冷却时间：`5.5 秒`

## 大招：Blizzard

- 技能类型：持续范围伤害 + 高频移速减益 Debuff
- 释放目标：以施法范围内敌方最密集区域为中心释放
- 效果：
  - 在目标点生成一个固定不跟随的暴风雪区域
  - 对区域内敌人持续造成范围伤害
  - 每次脉冲造成基于攻击力 `0.35` 倍率的范围伤害
  - 每次命中都会附加或刷新一次 `MoveSpeedModifier`
  - 单次减速幅度为 `-60%`
  - 单次减速持续 `1.2 秒`
  - 区域半径为 `5.5`
  - 持续 `5 秒`
  - 每 `1 秒` 脉冲一次
  - 区域固定在释放瞬间的位置，不跟随施法者移动
  - 大招的核心收益是范围控场和走位压制，不是靠总伤害单独完成收割
- 检测条件：
  - 本局尚未释放过大招
  - 优先要求施法距离 `7` 内能找到半径 `5.5` 范围内至少存在 `3` 名敌人的区域锚点
  - 若开战 `30 秒` 后仍未满足，则阈值降低为至少 `2` 名敌人
  - 若开战 `45 秒` 后仍未满足，则阈值降低为至少 `1` 名敌人
- 当前关键参数：
  - 施法距离：`7`
  - 区域半径：`5.5`
  - 脉冲伤害：基于攻击力 `0.35`
  - 区域持续时间：`5 秒`
  - 脉冲间隔：`1 秒`
  - 单次减速幅度：`-60%`
  - 单次减速持续时间：`1.2 秒`
- 当前关键约束：
  - 主动释放型大招本局仅能释放 `1 次`

## 落地约束

- 冰霜法师当前定位为“第二个法师扩展原型”，不替代 `Firemage` 作为首批范围爆发法师模板
- `Frost Burst` 和 `Blizzard` 应优先复用统一的“范围伤害 + 附加状态效果”表达，不应额外引入冰冻系专属底层
- 两个技能的减速都应直接复用 `MoveSpeedModifier` 的负 `magnitude` 语义，不新增单独的 `Slow` 状态类型
- `Blizzard` 应优先复用持续区域 + 周期脉冲载荷的统一技能结构；区域在释放后与施法者本体解耦，不因施法者后续移动、普攻或死亡而提前结束
- `Blizzard` 的重复命中应优先采用 `maxStacks = 1` + 刷新持续时间的方式维持高额减速，不应把移速无限叠到失控
- 冰霜法师的控制强度应明确停留在“减速压制”层级，不应偷偷扩成眩晕、冻结或击飞核心

## 设计边界

- 冰霜法师的核心职责是控场和限制敌方走位，不承担高爆发主力输出职责
- 不应把冰霜法师做成依赖连续硬控链吃饭的控制核心
- 不应把冰霜法师做成高机动切后排刺客式法师
- 不应把冰霜法师做成兼顾高续航或高承伤的前排法师
- 第一阶段下，冰霜法师优先验证“远程投射物 + 范围减速小技能 + 持续暴风雪区域大招”这套控场法师模板
