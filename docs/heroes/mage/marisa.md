# Marisa

## 基础信息

- 英文名：Marisa
- 中文名：雾雨魔理沙
- 开发代号：星帚魔女
- 英雄 ID：`mage_007_marisa`
- 英雄分类：法师
- 标签：远程、法术输出、弹幕、短距机动、直线爆发
- 核心定位：中后排高速弹幕法师，通过星屑弹持续压低敌方血线，用扫帚短距调整站位，最后用一局一次的巨大魔炮贯穿敌方阵型
- 默认 AI 模板：`mage_default`
- 当前定位补充：作为 `mage_006_mirrormage` 之后的第七个法师扩展原型，不替代 `Firemage` 的持续范围爆发，不替代 `Astromancer` 的延迟预判轰炸，不替代 `Lightningmage` 的高频小控，也不替代 `Mirrormage` 的战术分身；主要验证“远程高速星弹 + 短距安全位移 + 多方向弹幕 + 一局一次引导型直线魔炮”这条火力型魔女法师方向

说明：
- `法术输出` 在当前文档中是主题和定位标签；第一阶段战斗仍遵守统一伤害体系，不区分物理 / 法术伤害。
- 当前使用 `雾雨魔理沙` 作为英雄名与开发代号来源；若后续需要原创包装，可在不改变技能契约的前提下再改名。

## 角色主题

- 主题关键词：星星、扫帚、魔女帽、魔法弹幕、闪耀光束、活泼、贪玩、火力至上、横版弹幕感
- 角色气质：自信、活泼、有点鲁莽，倾向于用更大的火力解决问题
- 与 `Astromancer / 星术师` 的区分：
  - 星术师偏星象、落点预判、延迟轰炸和区域停留惩罚
  - 魔理沙偏扫帚、星弹、弹幕、直线魔炮和爽快爆发火力
  - 魔理沙不应依赖大范围延迟落点命中，也不应成为另一种星术师换皮

## 当前数值倾向

- 魔理沙应比传统法师更灵活一点，但不应拥有刺客级突进、无敌或后排切入能力
- 普攻节奏略快，单发伤害略低，用来体现“高速星弹持续压血”，不把普攻做成范围伤害
- 小技能主要提供短距拉开与短时间弹幕压制，伤害窗口可以可观，但不能兼具长距离切入、硬控或保命无敌
- 大招是她的核心招牌，应按持续纯伤害大招设计：满额命中应稳定击杀标准脆皮，重创中坚单位，并允许单独击杀完整吃满的满血坦克；但因为需要蓄力、持续引导、站位和路径命中，敌方应有打断或走位规避空间
- 魔理沙面对刺客贴脸、硬控打断、分散阵型或快速侧移时，应明显掉强

## 当前实现基线

- 最大生命：`290`
- 攻击力：`32`
- 防御力：`8`
- 攻速：`1.053`，约等于 `0.95 秒 / 次`
- 移速：`4.05`
- 暴击率：`10%`
- 暴击伤害：`1.6`
- 攻击距离：`8.8`

## 普攻类型：Star Shot / 星屑弹

- 技能类型：远程投射物普攻
- 释放目标：当前索敌目标；默认优先攻击最近的可攻击敌人，沿用法师和远程投射物英雄的共享索敌与受压后撤基线
- 效果：
  - 发射一枚小型星屑魔弹，对单个敌方目标造成基于攻击力 `0.78` 倍率的单体伤害
  - 星屑弹飞行速度较快，命中时播放小型星光碎裂表现
  - 普攻只承担稳定压血和弹幕节奏职责，不附带范围伤害、硬控、减速、标记或叠层
- 检测条件：
  - 当前存在合法敌方目标
  - 目标进入攻击范围
  - 普攻间隔转好
- 当前关键参数：
  - 普攻伤害倍率：基于攻击力 `0.78`
  - 攻击距离：`8.8`
  - 普攻间隔：`0.95 秒`
  - 投射物速度：`16`

## 小技能：Broom Burst / 星帚疾射

- 技能类型：自身短距安全位移 + 多方向星弹路径伤害
- 释放目标：自身
- 效果：
  - 魔理沙骑着扫帚进行一次短距滑行，优先朝远离最近近身威胁、偏向己方安全侧的合法落点移动
  - 位移距离比普通后撤更明显，目标是重新拉开输出距离或调整魔炮前的站位，不是切入敌方后排
  - 滑行期间和落点附近向多个方向发射星星，对短路径上的敌方英雄造成弹幕伤害
  - 每颗星星按短直线路径收集命中目标；同一目标在同一次 `Broom Burst` 中最多承受 `2` 次星弹命中
  - 单颗星星造成基于攻击力 `0.38` 倍率的伤害
  - 该技能不附带击退、眩晕、减速、不可选中或无敌
- 安全位移语义：
  - 落点必须通过共享合法落点规则校验，并受场地边界、最小间距和阵型侧约束
  - 默认不允许越过敌方前排进入敌方后排深处
  - 默认不允许把位移方向选成“朝敌方核心继续突进”
  - 若没有合法短距安全落点，本次技能应延后，不应强行穿人或越界
- 检测条件：
  - 小技能 CD 转好
  - 当前没有被硬控到无法施法
  - 当前存在合法敌方目标，且预计至少 `1` 颗星弹可命中敌方英雄
  - 若最近威胁敌人距离小于 `4.5`，显著提高释放优先级
  - 若当前完全没有合法安全落点，则不释放
- 当前关键参数：
  - 位移距离：`3.2` 世界单位
  - 位移持续时间：`0.42 秒`
  - 星弹数量：`6`
  - 单颗星弹路径长度：`4.8`
  - 单颗星弹路径宽度：`0.9`
  - 单颗星弹伤害倍率：基于攻击力 `0.38`
  - 同一目标单次技能最多命中：`2` 次
  - 冷却时间：`7 秒`

## 大招：Starlance Cannon / 星辉魔炮

- 中文名：星辉魔炮
- 技能类型：一局一次，蓄力引导型持续直线路径伤害
- 释放目标：优先选择从自身当前位置出发、预计命中敌方人数最多的一条直线路径；若没有理想路径，则回退到当前攻击目标方向
- 效果：
  - 魔理沙举起扫帚或魔法核心，短暂蓄力并聚集星光
  - 蓄力完成后向前方释放巨大星辉光束，持续一段时间
  - 光束按固定 tick 间隔对直线路径内所有敌方英雄造成伤害
  - 引导期间魔理沙不能移动、普攻或释放其他主动技能
  - 光束方向可以缓慢转动，但不能瞬间锁头追踪；敌方通过侧移、散开或打断引导可以降低吃到的 tick 数
  - 每次 tick 对路径内敌方英雄造成基于攻击力 `3.30` 倍率的伤害
  - 同一敌人在同一 tick 内最多被命中 `1` 次
  - 大招结束后进入短暂后摇
  - 该大招不附带眩晕、击退、禁疗、减防、不可选中或无敌
- 中断与资源语义：
  - 一局一次资源在蓄力开始时消耗
  - 蓄力或引导期间若魔理沙死亡、进入复活等待，或受到 `Stun / KnockUp` 等硬控，剩余蓄力或后续 tick 立即中断
  - 已经结算的伤害保留；未发生的 tick 不补偿
  - 中断也不返还本局大招资源
- 检测条件：
  - 本局尚未释放过大招
  - 当前没有被硬控到无法施法
  - 优先要求预计路径上至少可命中 `3` 名敌方英雄
  - 若开战 `35 秒` 后仍未满足，则阈值降低为至少 `2` 名敌方英雄
  - 若开战 `50 秒` 后仍未满足，则只要当前攻击目标方向上存在至少 `1` 名合法敌方目标即可释放
- 当前关键参数：
  - 蓄力时间：`0.85 秒`
  - 引导持续时间：`2.8 秒`
  - 后摇时间：`0.35 秒`
  - 路径长度：`20`
  - 路径宽度：`2.4`
  - tick 间隔：`0.35 秒`
  - 预计 tick 次数：`8`
  - 单 tick 伤害倍率：基于攻击力 `3.30`
  - 最大转向角速度：`35 度 / 秒`
- 当前关键约束：
  - 主动释放型大招本局仅能释放 `1` 次
  - 满额吃满伤害应按持续纯伤害大招看待：满血坦克完整吃满时应允许被单独击杀，但实战不应默认全额命中所有敌人

## 落地约束

- 魔理沙当前定位为“第七个法师扩展原型”，不替代现有任何法师模板；她的核心差异是短距扫帚机动与可被打断的直线持续魔炮
- `Star Shot` 应优先走 `BasicAttackData` 的远程投射物普攻配置；命中星光碎裂只是表现，不参与伤害判定
- `Broom Burst` 应优先复用共享位移 / reposition 能力和共享路径命中能力；如果实现层暂时没有多方向星弹配置，应先补共享多方向弹幕载荷或明确用短期近似方案标记，不要把弹幕写成魔理沙私有脚本
- `Broom Burst` 的位移落点必须由逻辑层判定，表现层只播放扫帚滑行和星弹，不反向决定落点、命中或是否穿越敌方阵型
- `Starlance Cannon` 应优先复用 `docs/planning/stage-01-hero-spec-decisions.md` 中的“引导型持续路径伤害能力”：蓄力、动作锁、路径 tick、缓慢转向、中断和一局一次资源都应是正式参数
- 魔炮命中应按逻辑路径宽度和长度收集目标，不依赖表现层光束 mesh、粒子碰撞或物理触发器决定实际伤害
- 魔炮每次 tick 的命中目标、伤害、中断原因和方向变化都应进入统一事件或调试日志，方便核对它是否符合“高伤害但可规避”的设计
- 表现层可以用更夸张的星光、扫帚尾迹和巨大光束，但不得把视觉宽度长期做得明显大于逻辑路径宽度，避免玩家误判命中范围

## 设计边界

- 魔理沙是中后排远程火力法师，不是前排坦克、刺客、治疗辅助或纯控制法师
- 不应把 `Star Shot` 做成范围普攻、连锁普攻或附带控制的机制主轴
- 不应把 `Broom Burst` 做成长距离冲刺、穿越敌方后排、不可选中、无敌或强控技能
- 不应让 `Broom Burst` 击退敌人或把敌人推开；它的自保来自自身短距调整，而不是控制对手
- 不应把 `Starlance Cannon` 做成全屏必中、瞬间无预警核爆或自动锁头持续追踪
- 不应给魔炮额外塞眩晕、击退、禁疗、减防等附加控制 / Debuff；它的价值集中在直线高伤害和站位压迫
- 不应让魔理沙拥有刺客级持续机动；她被近身、被打断、站位失误或敌方分散时，应存在明确反制空间
- 第一阶段下，魔理沙优先验证“远程高速投射物普攻 + 短距安全位移弹幕 + 一局一次可中断引导魔炮”这套弹幕火力法师模板
