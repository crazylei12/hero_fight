# Blazeroar

## 基础信息

- 英文名：Blazeroar
- 中文名：炽焰咆哮虎
- 英雄 ID：`tank_007_blazeroar`
- 英雄分类：坦克
- 标签：近战、承伤、控制、减攻、自我续航、路径冲锋
- 核心定位：扰乱型近战坦克，通过切换目标后的第一击短控打断敌方节奏，用出生 / 复活咆哮和主动后撤持续压低敌方攻击力，再用一局一次的自我治疗与路径冲锋重新压回前线
- 默认 AI 模板：`tank_default`
- 当前定位补充：作为 `tank_001_ironwall`、`tank_002_shieldwarden`、`tank_003_tidehunter`、`tank_004_mundo`、`tank_005_blastshield` 和 `tank_006_mammoth` 之后的第七个坦克扩展原型，不替代骑士的团队保护、护阵者的嘲讽反切入、潮汐猎人的范围反开、蒙多的纯自愈站场、爆破盾牌的反近战阵地或猛犸的主动开团；炽焰咆哮虎主要验证“目标切换首击眩晕 + 出生 / 复活全体减攻 + 主动后撤减攻 + 自疗冲锋”这条减攻扰乱型坦克方向

## 当前数值倾向

- 基础承伤面板应略低于纯站场坦克，避免出生 / 复活减攻、自身回血和路径冲锋叠加后过度耐打
- 攻击力保持坦克中低档，主要价值来自控制、减攻和冲锋路径扰乱，不靠普攻或大招完成稳定收割
- 移速可以略高于重型主坦，用来体现虎形前排的压迫感，但不应高到兼具稳定追击和无风险脱离
- 强度预算应优先放在“让敌方输出窗口变差”和“出生 / 复活后干扰战线”上，不额外补嘲讽、群体护盾、无敌或长时间团控

## 建议实现基线

- 最大生命：`520`
- 攻击力：`36`
- 防御力：`34`
- 攻速：`0.95`，约等于 `1.05 秒 / 次`
- 移速：`4.25`
- 暴击率：`5%`
- 暴击伤害：`1.5`
- 攻击距离：`1.8`

## 普攻类型

- 类型：近战瞬时命中普攻
- 释放目标：当前索敌目标；坦克默认优先接敌并黏住最近敌人，找不到时继续向敌方区域前压
- 效果：造成基于攻击力 `1.0` 倍率的单体伤害；若当前处于 `Blazing Roar` 的目标切换首击标记，则改为较低伤害并附加眩晕
- 检测条件：
  - 当前存在合法敌方目标
  - 目标进入攻击范围
  - 普攻间隔转好
- 当前关键参数：
  - 攻击距离：`1.8`
  - 普攻间隔：`1.05 秒`

## 小技能：Blazing Roar（炽焰咆哮）

- 技能类型：主动型小技能 + 被动监听组件 / 减攻 / 自身后撤 / 目标切换首击控制 / 出生与复活触发全体减攻
- 释放目标：主动释放时锁定当前敌方目标；被动首击作用于新切换到的当前目标；出生 / 复活触发效果作用于敌方全体英雄
- 被动效果一：目标切换首击
  - 当炽焰咆哮虎的当前目标从一个敌方英雄切换到另一个敌方英雄时，给自己的下一次普攻添加一次性首击标记
  - 下一次普攻若命中新目标，则本次普攻伤害降低为基于攻击力 `0.45` 倍率，并对目标附加 `Stun`
  - 首击标记只对这次目标切换后的新目标生效；若命中其他目标或目标失效，标记应被清理或等待下一次正式目标变化重新生成
  - 同一目标在短时间内反复切入切出不应无限刷眩晕；同一目标触发首击眩晕后，建议至少 `2.5 秒` 内不能再次因切回该目标触发
- 被动效果二：出生 / 复活咆哮
  - 每当炽焰咆哮虎在开局首次出生，或从死亡等待中复活时，对敌方全体英雄附加一次 `AttackPowerModifier`
  - 开局首次出生只触发 `1` 次；后续每次死亡后的复活事件都可以再次触发
  - 如果某个敌方英雄在该效果期间死亡，状态按统一死亡清空规则处理，复活后不继承
- 主动效果：反向退吼
  - 小技能 CD 转好后，炽焰咆哮虎朝当前目标的反方向后退一段距离
  - 后退不是长期逃跑状态，只是一次短距离 reposition，结束后继续按正常坦克 AI 重新决策
  - 同时对当前目标附加一次 `AttackPowerModifier`，压低其短时间输出
  - 若当前没有合法目标，或反向后退落点无法在竞技场边界与最小间距规则内得到合法位置，本次不释放
- 检测条件：
  - 主动小技能 CD 转好
  - 当前存在合法敌方目标
  - 当前目标距离自己不超过 `4.5`
  - 自身当前生命值比例 `<= 70%`，或当前目标最近 `5 秒` 内对己方累计造成至少 `80` 点有效直接伤害
- 当前关键参数：
  - 首击伤害：基于攻击力 `0.45`
  - 首击眩晕持续时间：`0.65 秒`
  - 同目标首击触发间隔：`2.5 秒`
  - 出生 / 复活全体减攻：攻击力 `-18%`
  - 出生 / 复活减攻持续时间：`4 秒`
  - 主动后退距离：`3.0`
  - 主动后退时长：`0.25 秒`
  - 主动目标减攻：攻击力 `-30%`
  - 主动目标减攻持续时间：`4 秒`
  - 主动冷却时间：`8 秒`

## 大招：Inferno Rush（炎虎冲锋）

- 技能类型：主动型大招 / 自我直接治疗 / 目标方向冲锋 / 路径范围伤害
- 释放目标：当前敌方目标；若当前目标不合法，则按最近敌人回退策略选择一个合法敌方目标
- 效果：
  - 释放时先对自己结算一次直接治疗，治疗量为自身最大生命值的 `25%`
  - 随后朝目标方向发起冲锋，冲锋方向在释放瞬间锁定，不持续追踪目标后续移动
  - 冲锋路径上的每个敌方英雄最多受到一次路径伤害
  - 路径伤害基于炽焰咆哮虎攻击力结算，继续走统一伤害、暴击、防御和日志入口
  - 大招不默认附带眩晕、击飞、嘲讽、击退、无敌或不可选中；它的核心价值是残血续命后重新穿入战线并打乱敌方站位压力
- 检测条件：
  - 本局尚未释放过大招
  - 当前存在合法敌方目标或最近敌方目标
  - 优先要求自身生命值比例 `<= 45%`
  - 同时要求冲锋路径预计至少命中 `2` 名敌方英雄，或当前目标最近 `5 秒` 内对己方累计造成至少 `120` 点有效直接伤害
  - 若开战 `45 秒` 后仍未释放，则生命值门槛放宽为 `<= 65%`，且路径预计命中 `1` 名敌方英雄即可释放
- 当前关键参数：
  - 自我治疗：`25% 最大生命值`
  - 冲锋最大距离：`7.0`
  - 冲锋路径宽度：`2.2`
  - 冲锋时长：`0.45 秒`
  - 路径伤害：基于攻击力 `3.0`
- 当前关键约束：
  - 主动释放型大招本局仅能释放 `1 次`

## 落地约束

- `Blazing Roar` 虽然同时包含被动监听和主动释放，但仍然只占用唯一小技能位，不新增独立被动栏或第四个技能槽
- 目标切换首击应基于运行时当前目标 `runtimeId` 的实际变化生成一次性标记；AI 每次重新评估但目标未变时不应刷新标记
- 首击眩晕必须直接复用统一 `Stun` 状态语义；较低伤害应作为本次普攻的正式倍率覆写，不要在命中后私下补扣或补回伤害
- 出生 / 复活咆哮应监听统一开局出生与复活事件；不应在召唤、分身创建或表现层动画事件里触发
- 所有减攻都必须使用统一 `AttackPowerModifier`，`magnitude` 按百分比小数语义落地；不要新增“咆哮减攻”专属属性通道
- 主动后退应复用共享 reposition / forced movement 能力，并遵守竞技场边界、最小间距和动作锁规则；后退期间不提供无敌、不可选中或免控
- `Inferno Rush` 的 `25% 最大生命值` 治疗必须走统一治疗入口，并正常受到目标侧 `HealTakenModifier / 受治疗率修正` 影响；禁疗不应被大招绕过
- 冲锋路径伤害应优先表达为“锁定方向 + 路径命中收集 + 每目标最多命中一次”的共享路径技能能力，不要依赖物理碰撞体实时扫线决定实际伤害
- 如果炽焰咆哮虎在冲锋前摇或冲锋过程中死亡，未完成的位移和未结算的路径命中应停止；已经完成的治疗和已结算命中不倒回
- 事件和日志至少要能明确记录：目标切换首击标记生成、首击是否命中并眩晕、出生 / 复活咆哮命中敌人数、主动后退落点、目标减攻、大招治疗量、冲锋路径命中人数和每个目标的实际伤害

## 设计边界

- 炽焰咆哮虎的核心职责是前排扰乱、减攻压制，以及出生 / 复活后压低敌方输出窗口，不承担爆发收割或团队保护主职责
- 不应把目标切换首击做成高伤害眩晕；它必须保留“伤害较低、换取短控”的代价
- 不应让目标切换首击因为默认索敌频繁重评估而变成近乎常驻眩晕链；同目标触发间隔和实际目标变化判断必须保留
- 不应把出生 / 复活咆哮扩成治疗友军、全体护盾、全体恐惧或改变复活规则的团队技能；它只负责短时降低敌方攻击力
- 不应把主动后退扩成长时间逃跑 AI、不可选中脱战或免疫控制窗口；它只是一次短距离重定位和单体减攻
- 不应把 `Inferno Rush` 扩成带群体眩晕、击飞、嘲讽或高额击退的万能开团大招；它已经同时拥有自疗、位移和路径伤害
- 第一阶段下，炽焰咆哮虎优先验证“近战普攻 + 目标切换首击眩晕 + 出生 / 复活全体减攻 + 主动后撤减攻 + 一局一次自疗冲锋”这套减攻扰乱型坦克模板
