# Mammoth

## 基础信息

- 英文名：Mammoth
- 中文名：猛犸
- 英雄 ID：`tank_006_mammoth`
- 英雄分类：坦克
- 标签：近战、承伤、开团、强制位移、聚怪、击退、击飞
- 核心定位：主动开团型近战坦克，先闪烁到施法距离内最远敌人的身后制造包夹位置，再用击退型回顶把敌人沿回程方向顶回己方前线；大招优先在闪烁落位后插入使用，把身前范围内敌人聚拢并击飞，再衔接小技能的回顶冲锋形成群体推挤窗口
- 默认 AI 模板：`tank_default`
- 当前定位补充：作为 `tank_001_ironwall`、`tank_002_shieldwarden`、`tank_003_tidehunter`、`tank_004_mundo` 和 `tank_005_blastshield` 之后的第六个坦克扩展原型，不替代骑士的团队保护、护阵者的嘲讽反切入、潮汐猎人的范围反开、蒙多的纯自愈站场或爆破盾牌的反近战阵地；猛犸主要验证“最远敌人身后落位 + 分段小技能插入大招 + 路径群体推挤”这条主动开团坦克方向

## 当前数值倾向

- 基础承伤面板应接近主坦，但略低于纯站场坦克；猛犸的强度预算主要放在开团、聚怪和强制位移上
- 攻击力偏低到中等，常态普攻不是主要威胁
- 移速保持坦克偏慢，不能同时拥有高机动开团和优秀常态追击
- 小技能与大招的组合很强，因此不应额外给高额伤害、护盾、无敌或长时间控制链

## 建议实现基线

- 最大生命：`530`
- 攻击力：`36`
- 防御力：`34`
- 攻速：`0.88`，约等于 `1.14 秒 / 次`
- 移速：`4.05`
- 暴击率：`5%`
- 暴击伤害：`1.5`
- 攻击距离：`1.9`

## 普攻类型

- 类型：近战瞬时命中普攻
- 释放目标：当前索敌目标；坦克默认优先接敌并黏住最近敌人，找不到时继续向敌方区域前压
- 效果：造成基于攻击力 `1.0` 倍率的单体伤害；普攻本体不附带控制、强制位移或聚怪
- 检测条件：
  - 当前存在合法敌方目标
  - 目标进入攻击范围
  - 普攻间隔转好
- 当前关键参数：
  - 攻击距离：`1.9`
  - 普攻间隔：`1.14 秒`

## 核心连携：Blink -> Quake -> Skewer

- 猛犸的小技能是一个分段动作：`闪烁落位 -> 可选大招插入窗口 -> 回顶冲锋`
- 闪烁落位后，若大招资源尚未使用且当前位置能覆盖到足够敌人，AI 应显著提高大招释放优先级
- 大招成功释放后，小技能继续执行后续的回顶冲锋，把已经聚集到猛犸身前并处于击飞控制中的敌人一起顶回去
- 若大招条件不满足、资源已消耗或被团队级大招策略延后，小技能不等待，直接执行回顶冲锋
- 这套连携不新增第四个技能槽；小技能仍是一个主动小技能，大招仍是一局一次主动大招

## 小技能：Backhaul Skewer（回顶冲锋）

- 技能类型：主动型小技能 / 最远敌人身后落位 / 分段冲锋 / 路径群体击退
- 释放目标：施法距离内当前距离猛犸最远的合法敌方英雄；若距离并列，优先当前绝对生命值更低者，再比较当前生命值比例
- 效果：
  - 施法开始时快照猛犸当前位置、目标当前位置和目标合法性
  - 猛犸闪烁到目标远离自己施法前位置的一侧，落在目标身后的合法近身点
  - 闪烁落位后锁定一条“回顶方向”：从当前落点指向猛犸施法前位置
  - 闪烁后进入一个很短的大招插入检查点；若 `Mammoth Quake` 在此时成功释放，则等大招聚拢与击飞结算后继续冲锋
  - 随后猛犸沿回顶方向冲锋，把初始目标顶着往回推；这段回顶在规则上视为一次击退型强制位移
  - 回顶路径上的其他敌方英雄如果进入路径宽度，也会加入同一次击退集合，被一起顶回去
  - 每个被推动目标受到一次基于攻击力 `0.65` 倍率的直接伤害
  - 击退位移期间，这些目标不执行自发移动、普攻或主动施法；该行为来自共享强制位移语义，不额外附加眩晕
  - 被推动目标最终分配在猛犸冲锋终点前方的合法扇形落点中，避免多人堆叠到同一个坐标
- 检测条件：
  - 小技能 CD 转好
  - 施法距离内至少存在 `1` 名合法敌方英雄
  - 目标与猛犸当前距离不低于 `5.5`，避免近距离反复闪现造成读图混乱
  - 目标身后存在合法落点；若找不到合法落点，本次不释放
  - 若回顶路径预计能推动 `2` 名及以上敌方英雄，显著提高释放优先级
- 当前关键参数：
  - 施法距离：`10.5`
  - 目标身后落点距离：`1.15`
  - 回顶最大距离：`7.0`
  - 路径宽度：`2.4`
  - 强制位移时长：`0.55 秒`
  - 推挤伤害：基于攻击力 `0.65`
  - 大招插入检查点：`0.25 秒` 内完成逻辑判定
  - 冷却时间：`8 秒`

## 大招：Mammoth Quake（猛犸震聚）

- 技能类型：主动型大招 / 范围聚拢 / 身前落点分配 / 击飞
- 释放目标：以猛犸为中心收集一定范围内的敌方英雄，并按猛犸当前朝向或回顶方向把目标聚集到自己身前
- 效果：
  - 释放时记录猛犸当前位置和正面方向
  - 若大招在 `Backhaul Skewer` 的插入窗口内释放，正面方向固定使用小技能锁定的回顶方向
  - 若大招独立释放，正面方向默认朝当前目标；若当前目标不合法，则朝敌方单位密集方向或敌方半场方向
  - 对范围内敌方英雄分别结算一次朝猛犸身前的强制位移
  - 系统优先把目标分配到猛犸身前 `1.4 ~ 2.8` 的合法扇形落点，而不是把所有敌人叠在同一点
  - 聚拢完成后，对所有被聚拢目标附加 `KnockUp`
  - 大招不造成直接伤害，不附带护盾、减伤、嘲讽、禁疗、眩晕或无敌；核心收益是聚怪、击飞打断节奏，并为后续回顶冲锋制造多人击退窗口
- 检测条件：
  - 本局尚未释放过大招
  - 若处于 `Backhaul Skewer` 的闪烁后插入窗口内，只要自身 `6.5` 范围内至少存在 `2` 名敌方英雄，就显著提高释放优先级并优先插入
  - 若不处于小技能插入窗口，优先要求自身 `6.5` 范围内至少存在 `3` 名敌方英雄
  - 若开战 `45 秒` 后仍未释放，则独立释放门槛放宽为自身 `6.5` 范围内至少 `2` 名敌方英雄
  - 若当前只能命中 `1` 名敌方英雄，默认不释放，避免把一局一次大招浪费成单体击飞
- 当前关键参数：
  - 聚拢半径：`6.5`
  - 身前落点扇形距离：`1.4 ~ 2.8`
  - 聚拢强制位移时长：`0.30 秒`
  - 击飞持续时间：`1.25 秒`
- 当前关键约束：
  - 主动释放型大招本局仅能释放 `1 次`

## 落地约束

- 猛犸当前定位为“第六个坦克扩展原型”，不替代既有坦克方向；后续实现时不要为了提高稳定性把他扩成全能前排
- `Backhaul Skewer` 的目标选择应优先复用或补齐共享 `施法者指定范围内最远的敌方英雄` 目标策略，不要把“最远敌人”偷换成“最后排”“最低血量”或固定英雄私有排序
- 闪烁到目标身后应优先复用共享 `reposition / blink` 能力；闪烁只负责落位，不提供无敌、不可选中、免伤、免控或死亡防止
- 身后落点必须以施法前猛犸和目标的相对方向计算；不要在目标已经被移动后再用新位置反推“身后”
- 小技能的“大招插入窗口”必须遵守 `docs/planning/stage-01-hero-spec-decisions.md` 中的分段动作插入规则：只允许插入猛犸自己的大招，且大招仍消耗正式一局一次资源
- 回顶冲锋应优先复用共享的 `路径推挤型强制位移能力`：用路径扫掠收集目标、统一击退型强制位移和合法落点分配表达，不要依赖物理碰撞体实时互推来决定谁被顶走
- 路径中追加加入的敌人同一次冲锋最多被收集 `1` 次；已经加入推挤集合的目标不应因路径重叠重复受伤或重复位移
- 多目标最终落点必须通过共享合法落点分配或最小间距收束处理，不能把被聚拢 / 被推挤的敌人直接塞到一个点
- `Mammoth Quake` 的聚拢应复用或扩展屠夫、魔剑客这类强制位移已有的多目标落点分配思路；聚拢本身不新增“猛犸压制”“被顶状态”等专属底层状态
- 大招击飞必须直接复用统一 `KnockUp` 状态语义；聚拢强制位移和击飞是两个明确效果，不应把水平位移塞进击飞状态里
- 如果猛犸在闪烁后、回顶前死亡，后续回顶冲锋取消；如果大招已经完成结算，已经施加的聚拢和击飞不倒回
- 如果猛犸在闪烁后、回顶前受到硬控，回顶冲锋应被打断或延后规则必须显式实现；第一版建议直接取消后续冲锋并消耗本次小技能 CD
- 事件和日志至少要能明确记录：小技能目标、闪烁落点、是否触发大招插入、大招聚拢人数、回顶路径收集人数、每个目标的最终落点与实际位移距离

## 设计边界

- 猛犸的核心职责是主动开团、聚怪和把敌方阵型顶回己方前线，不承担高爆发收割职责
- 不应把猛犸做成带护盾、治疗、无敌、不可选中、分担伤害和高额自愈的全能生存坦克
- 不应把 `Backhaul Skewer` 扩成全屏必中闪现；施法距离、身后落点和最小距离门槛必须保留反制空间
- 不应让回顶冲锋无视边界、最小间距或目标合法性，把敌方单位强行堆叠到不可读位置
- 不应把 `Mammoth Quake` 扩成全屏聚怪或超长击飞控制；它应是闪烁落位后的强团控窗口，而不是任何时间都能稳定清场的万能大招
- 不应让大招在只命中单个目标时频繁释放；这个英雄的设计上限来自“聚集多人后顶回”，不是单点抓人
- 猛犸闪烁到敌阵后若被立即集火或控制，应有失败风险；不要给他额外补短时霸体、免控或免伤来抹掉这个风险
- 第一阶段下，猛犸优先验证“近战普攻 + 分段闪烁回顶击退小技能 + 身前聚拢击飞大招 + 大招插入后群体击退”的主动开团坦克模板
