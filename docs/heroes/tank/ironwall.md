# Ironwall

## 基础信息

- 英文名：Ironwall
- 中文名：骑士
- 英雄 ID：`tank_001_ironwall`
- 英雄分类：坦克
- 标签：近战、承伤、控制、团队增益
- 核心定位：前排近战控制坦克，优先顶住当前交战目标，用单点硬控打断敌方前压，再用全队防御增益稳住团战
- 默认 AI 模板：`tank_default`

## 普攻类型

- 类型：近战瞬时命中普攻
- 释放目标：当前索敌目标；坦克默认优先接敌并黏住最近敌人，找不到时继续向敌方区域前压
- 效果：造成基于攻击力 `1.0` 倍率的单体伤害
- 检测条件：
  - 当前存在合法敌方目标
  - 目标进入攻击范围
  - 普攻间隔转好
- 当前关键参数：
  - 攻击距离：`1.8`
  - 普攻间隔：`1.11 秒`

## 小技能：Shield Bash

- 技能类型：单体控制
- 释放目标：当前目标
- 效果：
  - 对当前目标造成基于攻击力 `0.8` 倍率的单体伤害
  - 同时附加 `1 秒` 眩晕
  - 当前参数下更偏向“抢先打断前排接敌节奏”的近身控制技能
- 检测条件：
  - 小技能 CD 转好
  - 当前存在合法敌方目标
  - 目标进入施法距离 `1.8`
- 当前关键参数：
  - 伤害倍率：基于攻击力 `0.8`
  - 眩晕持续时间：`1 秒`
  - 冷却时间：`8 秒`

## 大招：Iron Oath

- 技能类型：全体增益 Buff / 属性上升
- 释放目标：全体友方英雄，包括自己
- 效果：
  - 立即为全体友军附加统一的防御力提升状态
  - Buff 直接附着在友军单位身上，不依赖固定区域，会随友军后续移动持续生效
  - 当前参数下更偏向“团队减伤窗口”，不提供护盾、治疗或无敌
- 检测条件：
  - 大招 CD 转好
  - 当前至少 `3` 名存活友军距离任意敌方英雄 `6` 以内
  - 若开战 `30 秒` 后仍未满足，则低血量人数阈值降低为至少 `1` 名
  - 若开战 `45 秒` 后仍未满足，则只要求至少 `2` 名存活友军已进入上述交战距离
- 当前关键参数：
  - 防御力提升：`+40`
  - 持续时间：`8 秒`
  - 冷却时间：`16 秒`

## 设计边界

- 骑士的核心职责是前排承伤、限制推进和团队保护，不承担爆发收割职责
- 不应把骑士做成靠高机动切后排的突进型英雄
- 不应把骑士做成以大范围连续硬控为主的控制核心
- 不应把骑士的大招扩成护盾、治疗或无敌类全能保命技能
- 第一阶段下，骑士优先验证“近战瞬时普攻 + 当前目标眩晕 + 全队防御 Buff 大招”这套坦克模板
