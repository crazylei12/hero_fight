# Bladesman

## 基础信息

- 英文名：Bladesman
- 中文名：剑客
- 英雄 ID：`warrior_002_bladesman`
- 英雄分类：战士
- 标签：近战、破甲、突进、爆发
- 核心定位：近战破阵战士，通过慢速重斩普攻和单体破甲小技能压低前排或被集火目标的承伤能力，再用高速直线穿阵大招切开敌方阵型
- 默认 AI 模板：`warrior_default`
- 当前定位补充：作为 `warrior_001_skybreaker` 之后的第二个战士扩展原型，不替代格斗家的“护盾冲锋 + 范围击飞开团”模板，主要验证“重型近战普攻 + 单体减防 + 高速直线穿阵大招”这条更偏破阵收口的战士方向

## 当前数值倾向

- 生命值：中等，能够站前排对拼，但不以纯承伤见长
- 防御力：中等，略低于偏承伤型前排
- 攻击力：高，单次命中威胁应明显高于 `Skybreaker`
- 攻速：偏低，用更慢的出手换更重的单次打击
- 移速：中高，需要能稳定贴住前排并找出穿阵角度
- 攻击距离：标准近战略长，形成“刀锋多探半步”的接敌体验

## 建议实现基线

- 最大生命：`410`
- 攻击力：`46`
- 防御力：`22`
- 攻速：`0.892857`，约等于 `1.12 秒 / 次`
- 移速：`4.5`
- 暴击率：`12%`
- 暴击伤害：`1.65`
- 攻击距离：`1.9`

## 普攻类型

- 类型：近战瞬时命中普攻
- 释放目标：当前索敌目标；战士默认优先接敌并黏住最近敌人，当前目标失效后继续追击最近合法敌人
- 效果：造成基于攻击力 `1.2` 倍率的单体伤害；普攻节奏偏慢，但每次命中威胁较重
- 检测条件：
  - 当前存在合法敌方目标
  - 目标进入攻击范围
  - 普攻间隔转好
- 当前关键参数：
  - 攻击距离：`1.9`
  - 普攻间隔：`1.12 秒`

## 小技能：裂甲斩

- 技能类型：单体伤害 + 防御减益 Debuff
- 释放目标：当前目标；若当前目标不合法或超出施法距离，则按共享 `NearestEnemy` 回退策略改抓最近合法敌人
- 效果：
  - 对目标造成一次基于攻击力 `1.35` 倍率的单体伤害
  - 同时附加一次 `DefenseModifier`
  - 需注意，需要先附加减防御状态，再结算伤害；
  - 减防幅度为 `-25%`
  - 减防持续 `4 秒`
  - 技能核心收益是先把前排或当前被集火目标“切开”，让后续普攻和大招更容易打穿
- 检测条件：
  - 小技能 CD 转好
  - 施法距离 `2.4` 内存在合法敌方目标
- 当前关键参数：
  - 施法距离：`2.4`
  - 直接伤害：基于攻击力 `1.35`
  - 减防幅度：`-25%`
  - 减防持续时间：`4 秒`
  - 冷却时间：`6.5 秒`

## 大招：飞燕断空

- 技能类型：直线高速突进 + 路径伤害
- 释放目标：锁定当前目标当前位置，沿施法者指向该目标的方向进行一次高速直线穿切；若当前目标不合法，则本次不释放
- 效果：
  - 剑客沿固定方向高速突进一段距离
  - 对路径条带上的所有敌方英雄造成一次基于攻击力 `8` 倍率的伤害
  - 路径命中的每个敌人只会在本次大招中受伤 `1 次`
  - 施法开始后路径方向锁定；即使原目标在突进过程中死亡、位移或失去直接目标合法性，剑客仍会沿既定方向完成穿切
  - 大招核心收益是“沿一条线快速切开敌阵”，既能压穿前排密集区，也能在目标已经被破甲后完成强力收口
- 检测条件：
  - 本局尚未释放过大招
  - 当前目标位于启动判定距离 `6.5` 内
  - 优先要求当前目标方向的穿切路径上至少可命中 `2` 名敌方英雄
  - 若开战 `40 秒` 后仍未满足双目标窗口，则只要当前目标方向路径上存在合法敌方目标即可释放
- 当前关键参数：
  - 启动判定距离：`6.5`
  - 穿切距离：`16`
  - 路径宽度：`4`
  - 穿切时长：`0.22 秒`
  - 路径伤害：基于攻击力 `8`
- 当前关键约束：
  - 主动释放型大招本局仅能释放 `1 次`

## 落地约束

- 剑客当前定位为“第二个战士扩展原型”，不替代 `Skybreaker` 的近身开团控制模板
- `裂甲斩` 必须直接复用统一的“直接伤害 + 防御减益状态”组合，不应额外创建剑客专属破甲底层
- `裂甲斩` 的减防应走正式 `DefenseModifier` 状态语义，方便后续与统一状态图标、日志和数值结算保持一致
- `飞燕断空` 应优先复用统一的“施法者自身高速位移 + 路径收集敌人 + 对路径目标结算一次伤害”能力；如果现有技能模板不足，应先补共享线性路径命中能力，不要把它写成剑客孤立特例
- 大招的路径方向应在施法开始时锁定，后续不继续追踪目标当前位置，避免把它实现成贴脸追踪刺杀技能
- 大招造成的是路径效果；命中判定应按统一的路径或范围命中规则收集敌人，不要额外要求这些敌人在命中瞬间仍是“当前直接目标”
- `飞燕断空` 有明显位移过程，表现层应优先沿用共享 `ForcedMovement` 冲锋语义和 `DashChargeTrail` 路径，再叠加细长刀光或斩线反馈，不应落成圆形爆炸或持续区域技能
- 表现重点应是“快速切开一条线”，而不是在终点做大范围停留特效；终点命中爆点可以有，但不应盖过整段直线穿切的阅读重点

## 设计边界

- 剑客的核心职责是破甲、切线和收口，不承担坦克式长期承伤职责
- 不应把剑客做成带稳定硬控链的前排开团战士，那会与 `Skybreaker` 的职责明显重叠
- 不应把剑客做成拥有多段追踪位移、瞬移或长时间不可选中的刺客式切后排角色
- 不应把 `裂甲斩` 扩成高倍率爆发、长时减防和额外硬控同时兼具的全能小技能
- 不应把 `飞燕断空` 扩成全屏超长位移、无视站位的必杀技能；它仍应强调“找准线、切穿线”的战士式进攻窗口
- 第一阶段下，剑客优先验证“慢速重斩普攻 + 单体破甲 + 一局一次高速直线穿阵大招”这套破阵型战士模板
