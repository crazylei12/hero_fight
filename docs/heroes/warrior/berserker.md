# Berserker

## 基础信息

- 英文名：Berserker
- 中文名：狂战士
- 英雄 ID：`warrior_003_berserker`
- 英雄分类：战士
- 标签：近战、持续输出、残血反打、自我强化、吸血
- 核心定位：前排缠斗战士，不靠先手控制开团，而是通过越残越痛的被动逼迫对手在“放着不管”和“硬着头皮打到底”之间做选择；一旦被压进危险血线，就能靠被动吸血配合大招的狂暴强化把换血重新拉回自己擅长的方向
- 默认 AI 模板：`warrior_default`
- 当前定位补充：作为 `warrior_002_bladesman` 之后的第三个战士扩展原型，不替代 `Skybreaker` 的开团控制模板，也不替代 `Bladesman` 的破甲切线模板，主要验证“被动型小技能 + 残血增伤 / 残血吸血 + 一局一次巨化狂暴”这条更偏反打与续战的战士方向

## 当前数值倾向

- 生命值：中高，需要能在危险血线前撑住一轮集火
- 防御力：中等偏低，不依赖高防硬吃整轮伤害
- 攻击力：中高，核心伤害高点来自被动与大招叠加窗口
- 攻速：中等，开大后应明显提速
- 移速：标准战士水平，靠持续追砍而不是长距离突进
- 攻击距离：标准近战偏长，保证巨化期间的贴身压迫更稳定

## 建议实现基线

- 最大生命：`440`
- 攻击力：`42`
- 防御力：`18`
- 攻速：`1.0`，约等于 `1.00 秒 / 次`
- 移速：`4.45`
- 暴击率：`8%`
- 暴击伤害：`1.6`
- 攻击距离：`1.85`

## 普攻类型

- 类型：近战瞬时命中普攻
- 释放目标：当前索敌目标；战士默认优先接敌并黏住最近敌人，当前目标失效后继续追击最近合法敌人
- 效果：造成基于攻击力 `1.0` 倍率的单体伤害；常态普攻不额外附带控制或 Debuff，强度重点留给被动残血增伤和大招爆发窗口
- 检测条件：
  - 当前存在合法敌方目标
  - 目标进入攻击范围
  - 普攻间隔转好
- 当前关键参数：
  - 攻击距离：`1.85`
  - 普攻间隔：`1.00 秒`

## 小技能：Bloodfury

- 技能类型：被动型小技能 / 动态攻击力增益
- 释放目标：自己；常驻生效，无主动施放动作
- 效果：
  - 根据当前已损失生命值比例实时提升攻击力
  - 攻击力加成为 `已损失生命值比例 * 0.6`
  - 加成上限为 `+60%`
  - 例如：生命值为 `70%` 时约获得 `+18%` 攻击力；生命值为 `40%` 时约 `+36%`；生命值为 `10%` 时约 `+54%`
  - 当当前生命值比例 `低于 40%` 时，额外获得 `35%` 吸血
  - 该吸血只看当前生命值，不保留“本局曾掉到多低”的历史记忆；血线回到 `40%` 及以上后立即失效
  - 该加成只按当前生命值即时计算，不保留“本局曾掉到多低”的历史记忆
- 检测 / 生效条件：
  - 狂战士处于存活状态
  - 当前生命值比例发生变化时应立刻刷新加成
  - 死亡后失效，复活后按复活时当前生命值重新计算
- 当前关键参数：
  - 缺血转攻击力系数：`0.6`
  - 攻击力加成上限：`+60%`
  - 吸血触发血线：`< 40%`
  - 吸血比例：`35%`
  - 生效方式：`当前生命值比例实时刷新`

## 大招：Titan Rage

- 技能类型：自身增益 Buff / 动作覆写 / 表现巨化
- 释放目标：自己
- 效果：
  - 立即进入 `6 秒` 狂暴状态
  - 体型放大到约 `1.4x`，强化近战压迫感与读图识别
  - 同时为自己附加一次 `AttackPowerModifier`
  - 攻击力提升 `+25%`
  - 同时附加一次 `AttackSpeedModifier`
  - 攻速提升 `+40%`
  - 同时附加一次 `MoveSpeedModifier`
  - 移速提升 `+25%`
  - 同时附加一次 `DefenseModifier`
  - 防御力提升 `+25%`
  - 大招核心收益是把残血被动放大成一段更稳的追砍与反杀窗口，而不是提供无脑开场爆发
- 检测条件：
  - 本局尚未释放过大招
  - 当前存在合法敌方目标
  - 优先要求当前生命值比例不高于 `65%`
  - 若自身 `3.2` 范围内至少存在 `2` 名敌方英雄，则显著提高释放优先级
  - 若开战 `45 秒` 后仍未满足血线条件，只要当前目标进入 `2.6` 范围内即可释放
- 当前关键参数：
  - 持续时间：`6 秒`
  - 表现体型倍率：`1.4`
  - 攻击力提升：`+25%`
  - 攻速提升：`+40%`
  - 移速提升：`+25%`
  - 防御力提升：`+25%`
- 当前关键约束：
  - 主动释放型大招本局仅能释放 `1 次`

## 落地约束

- `Bloodfury` 是“被动型小技能”的正式样例，应优先复用共享的被动小技能配置与当前生命值比例监听，不要写成狂战士专属 `Update` 轮询
- `Bloodfury` 的攻击力加成必须继续走统一属性汇总入口；低血量吸血也应继续走统一的伤害转治疗入口，不要并行维护一套私有回复逻辑
- `Titan Rage` 的攻击力和攻速提升必须分别复用正式 `AttackPowerModifier` 与 `AttackSpeedModifier` 语义，不要散写进普攻伤害公式或攻击计时器特判
- `Titan Rage` 新增的移速和防御提升必须继续复用正式 `MoveSpeedModifier` 与 `DefenseModifier` 语义，不要散写进移动或伤害公式特判
- 体型变大当前只作为表现和可读性强化，不改变碰撞体、攻击距离、命中收集半径、body blocking、threat 语义或其他共享战斗规则
- 大招不应额外附带不可选中、减伤、霸体或冲锋位移；狂战士的翻盘能力应来自贴身输出和吸血回补，而不是额外保命特权
- 日志和事件至少要能明确记录：被动攻击力 / 吸血的刷新、大招开始与结束，以及吸血转化出的治疗量

## 设计边界

- 狂战士的核心职责是残血反打和持续缠斗，不承担坦克式长期承伤职责
- 不应把狂战士做成带稳定硬控链的先手开团战士，那会与 `Skybreaker` 的职责重叠
- 不应把狂战士做成依赖多段突进、瞬移或不可选中的切后排刺客
- 不应把 `Bloodfury` 扩成同时提升攻击力、防御力、攻速和减伤的全能被动；它当前只负责残血增伤和低血吸血
- 不应把 `Titan Rage` 扩成开大即满血、无视集火或长期续航失控的超模技能；它应是短时间高风险高回报的追砍与反扑窗口
- 第一阶段下，狂战士优先验证“近战普攻 + 被动残血增伤 / 低血吸血 + 一局一次巨化狂暴”这套战士模板
