# Chainbreaker

## 基础信息

- 英文名：Chainbreaker
- 中文名：破链者
- 英雄 ID：`warrior_006_chainbreaker`
- 英雄分类：战士
- 标签：近战、反控制、持续输出、怒链、自我解控
- 核心定位：正面持续作战的反控制战士，不主动承担先手开团职责，而是在敌方用减速、眩晕、击飞、击退或嘲讽限制他时，把这些限制转化成攻速、攻击力和一次反击斩击
- 默认 AI 模板：`warrior_default`
- 当前定位补充：作为 `warrior_005_trollwarlord` 之后的第六个战士扩展原型，不替代 `Skybreaker` 的开团控制、`Berserker` 的残血续战、`Spellblade` 的阵型压缩或 `TrollWarlord` 的单挑攻速叠层；破链者主要验证“受限状态事件监听 + 自我负面清除 + 短时控制转化窗口”这条反控制战士方向

## 当前数值倾向

- 生命值：中等偏高，需要能站在正面吃到控制后仍有反击窗口
- 防御力：中等，不靠高防或减伤承担主坦职责
- 攻击力：中等偏高，核心爆点来自怒链消耗和大招期间的控制转化
- 攻速：中等偏慢，避免无怒链时也拥有过高持续输出
- 移速：标准战士水平，不靠主动突进开团
- 攻击距离：标准近战偏长，配合重斧读图但不突破近战定位
- 暴击收益：偏低，输出差异应来自怒链层数而不是随机暴击

## 建议实现基线

- 最大生命：`430`
- 攻击力：`41`
- 防御力：`20`
- 攻速：`0.925926`，约等于 `1.08 秒 / 次`
- 移速：`4.35`
- 暴击率：`8%`
- 暴击伤害：`1.6`
- 攻击距离：`1.85`

## 普攻类型

- 类型：近战瞬时命中普攻
- 释放目标：当前索敌目标；战士默认优先接敌并持续攻击最近合法敌人，当前目标失效后重新追击最近敌人
- 效果：使用重斧造成基于攻击力 `1.0` 倍率的单体伤害
- 检测条件：
  - 当前存在合法敌方目标
  - 目标进入攻击范围
  - 普攻间隔转好
- 当前关键参数：
  - 攻击距离：`1.85`
  - 普攻间隔：`1.08 秒`

## 核心机制：Rage Chain / 怒链

- 机制归属：`Break Free` 的常驻监听部分，仍占用破链者唯一的小技能位，不额外新增独立被动栏
- 触发条件：
  - 破链者被敌方成功施加 `眩晕`、`击飞`、`嘲讽` 或负数 `MoveSpeedModifier` 时，获得 `1` 层怒链
  - 破链者受到敌方负向强制位移请求，例如击退、拉拽、推开时，获得 `1` 层怒链
  - 大招 `Unchained Rampage` 期间，被转化掉的上述控制、减速或负向强制位移，也会获得 `1` 层怒链
  - 同一来源、同一技能或同一持续区域每 `0.8 秒` 最多触发 `1` 次怒链，避免高频减速区域瞬间堆满层数
- 层数效果：
  - 每层怒链提供 `+6%` 攻速
  - 最多 `5` 层，满层为 `+30%` 攻速
  - 每次获得怒链刷新怒链持续时间
- 当前关键参数：
  - 最大层数：`5`
  - 每层攻速提升：`+6%`
  - 持续时间：`6 秒`
  - 同源触发间隔：`0.8 秒`
- 清除规则：
  - 破链者死亡时清空怒链
  - 复活后不继承死亡前层数
  - `Break Free` 追加斩击成功触发时会消耗全部当前怒链

## 小技能：Break Free / 破缚

- 技能类型：主动型自我负面清除 + 短时攻速增益 + 怒链消耗斩击
- 释放目标：自己；追加斩击优先命中当前近战目标，若当前目标不合法则回退到 `2.4` 范围内最近合法敌人
- 效果：
  - 立即清除自身当前负向状态
  - 破缚可以在自身正处于 `眩晕`、`击飞` 或 `嘲讽` 时触发；这是仅服务自我解控的特殊触发入口，不代表所有主动技能都能绕过硬控门禁
  - 清除后获得 `2.5 秒` 短时攻速提升
  - 若当前拥有怒链层数，并且清除后存在合法近战目标，则消耗全部怒链并追加一次强力斩击
  - 追加斩击造成 `1.20 + 0.30 * 消耗怒链层数` 的攻击力倍率伤害
  - 若没有合法斩击目标，则只执行自我解控和攻速增益，不消耗怒链
  - 该技能不提供护盾、治疗、减伤、无敌或不可选中
- 检测条件：
  - 小技能 CD 转好即可释放
  - 不额外要求自身当前有负向状态、已有怒链层数或附近存在斩击目标
  - 若释放时没有可清除负向状态，则只获得短时攻速提升
  - 若释放时没有怒链层数或没有合法斩击目标，则不触发追加斩击，也不消耗怒链
- 当前关键参数：
  - 冷却时间：`8 秒`
  - 攻速提升：`+35%`
  - 攻速提升持续时间：`2.5 秒`
  - 追加斩击距离：`2.4`
  - 追加斩击基础倍率：基于攻击力 `1.20`
  - 每层怒链追加倍率：基于攻击力 `0.30`
  - 满层追加斩击倍率：基于攻击力 `2.70`

## 大招：Unchained Rampage / 无链狂袭

- 技能类型：主动型大招 / 开启自我解控 / 自身控制转化窗口 / 攻击力提升 / 结束范围斩击
- 释放目标：自己
- 效果：
  - 大招开启时，先立即解除自身当前控制类负向状态
  - 可解除范围包括自身当前的 `眩晕`、`击飞`、`嘲讽`、负数 `MoveSpeedModifier`，以及正在施加中的敌方负向强制位移限制
  - 无链狂袭可以在自身正处于 `眩晕`、`击飞` 或 `嘲讽` 时触发；这是仅服务大招开启解控的特殊触发入口，不代表所有大招都能绕过硬控门禁
  - 解控后，破链者进入 `5.5 秒` 无链狂袭状态
  - 状态持续期间，后续来自敌方的 `眩晕`、`击飞`、`嘲讽`、负数 `MoveSpeedModifier` 和负向强制位移请求会被转化
  - 被转化的控制、减速或强制位移不会附加对应行为阻断、移速下降或位置变化
  - 每次成功转化会获得 `1` 层怒链，并为大招期间额外提供 `+8%` 攻击力
  - 大招期间由转化获得的攻击力提升最多叠到 `+40%`
  - 大招内部记录本次持续期间累计转化出的怒链层数，最高按 `8` 层计入结束斩击
  - 大招结束时，以自身为圆心进行一次范围重斧斩击，对周围敌人造成 `1.40 + 0.35 * 本次大招累计怒链层数` 的攻击力倍率伤害
  - 若破链者在大招期间死亡，大招立即结束，且不触发结束范围斩击
  - 大招不阻止伤害、减防、减攻、禁疗、持续伤害、集火标记、被选中或范围命中
- 检测条件：
  - 本局尚未释放过大招
  - 当前存在合法敌方目标
  - 优先在自身已有至少 `2` 层怒链，或最近 `8 秒` 内至少受到过 `2` 次控制 / 减速 / 负向强制位移时释放
  - 优先要求当前目标距离不超过 `2.8`，且自身 `4.2` 范围内至少存在 `2` 名敌方英雄
  - 若开战 `45 秒` 后仍未释放，只要自身至少有 `1` 层怒链且当前目标进入 `2.4` 范围内即可释放
  - 若完全没有怒链且近期没有被限制，默认不主动释放，避免把大招浪费成普通增益
- 当前关键参数：
  - 持续时间：`5.5 秒`
  - 每次转化攻击力提升：`+8%`
  - 转化攻击力提升上限：`+40%`
  - 结束斩击半径：`3.4`
  - 结束斩击基础倍率：基于攻击力 `1.40`
  - 结束斩击每层追加倍率：基于攻击力 `0.35`
  - 结束斩击计入层数上限：`8`
- 当前关键约束：
  - 主动释放型大招本局仅能释放 `1 次`
  - 大招持续期间不暂停死亡、复活、技能 CD 或战斗计时

## 落地约束

- 破链者的怒链监听应优先走共享的“受限状态事件驱动的自我计数与转化”能力，不要写成 `ChainbreakerOnly` 的私有状态表
- 怒链提供的攻速提升必须复用正式 `AttackSpeedModifier` 或等价的统一属性修正入口，不要直接改写普攻计时器内部常量
- `Break Free` 的自我负面清除必须复用状态系统的负向状态过滤语义；它不是通用驱散，不能清除友军状态、敌方状态、复活等待或已经结算的伤害 / 位移结果
- `Break Free` 允许硬控期间触发这一点必须被限制在自我解控技能上，不应让所有主动技能都能绕过 `CanCastSkills` 门禁
- `Break Free` 的追加斩击应走统一单体伤害结算与事件日志；消耗层数、斩击目标和实际伤害都要能被日志追踪
- `Unchained Rampage` 开启瞬间的自我解控只清除控制类负向限制，不应顺手清除减防、减攻、禁疗、持续伤害、集火标记或其他未声明 Debuff
- `Unchained Rampage` 后续持续部分应优先复用 `控制转化窗口` 的窄语义：只转化后续控制、减速和敌方负向强制位移，不提供无敌、不可选中、减伤、护盾或死亡防止
- 负向强制位移被转化时，需要在正式位移生效前有一个统一拦截 / 转化入口；不要在表现层靠“把人拉回来”假装规则没有生效
- 同源触发频率限制必须生效，尤其要防止冰法暴风雪这类高频减速区域让怒链和攻击力提升瞬间失控
- 大招结束斩击应走统一范围伤害收集，不应额外要求敌人是当前直接目标；但也不应命中不可合法受范围伤害的目标
- 事件和日志至少要能明确记录：怒链获得、怒链刷新、怒链消耗、负面状态被清除、控制被转化、大招攻击力提升、大招结束斩击命中与伤害

## 设计边界

- 破链者的核心职责是把敌方控制转化为输出，不承担主坦克式长期承伤职责
- 不应把破链者做成纯免控坦克；他没有常驻霸体，也不应获得高额减伤、护盾、治疗、无敌或不可选中
- 不应把破链者做成主动开团战士；他没有冲锋、群体嘲讽、先手大范围击飞或稳定聚怪
- 不应让怒链在没有敌方限制的战斗里快速堆满；对无控制阵容时，破链者应只是一个中等强度的近战战士
- 不应把 `Break Free` 扩成通用驱散或团队解控技能；它只解决破链者自己被限制后的反击窗口
- 不应让 `Unchained Rampage` 开启解控或后续转化覆盖所有 Debuff；减防、减攻、禁疗、持续伤害和集火标记等仍应成为有效反制
- 不应让大招结束斩击在破链者死亡后仍触发，否则会削弱集火击杀这一反制点
- 第一阶段下，破链者优先验证“近战普攻 + 受限状态计数 + 自我解控消耗层数 + 一局一次控制转化反打”这套反控制战士模板
