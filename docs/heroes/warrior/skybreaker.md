# Skybreaker

## 基础信息

- 英文名：Skybreaker
- 中文名：格斗家
- 英雄 ID：`warrior_001_skybreaker`
- 英雄分类：战士
- 标签：近战、突进、控制、生存
- 核心定位：前排近战开团战士，依靠短距离冲锋抢先切入，用护盾顶住第一轮交换，再用近身范围击飞打乱敌方站位
- 默认 AI 模板：`warrior_default`

## 普攻类型

- 类型：近战瞬时命中普攻
- 释放目标：当前索敌目标；战士默认优先接敌并黏住最近敌人，找不到时继续向敌方区域前压
- 效果：造成基于攻击力 `1.0` 倍率的单体伤害
- 检测条件：
  - 当前存在合法敌方目标
  - 目标进入攻击范围
  - 普攻间隔转好
- 当前关键参数：
  - 攻击距离：`1.7`
  - 普攻间隔：`0.91 秒`

## 小技能：Breaker Rush

- 技能类型：单体突进 + 自身护盾 + 路径击飞
- 释放目标：施法距离内的单个敌方英雄，默认优先当前目标，若当前目标不合法则退回最近敌人
- 效果：
  - 施放瞬间先为自己附加一个短持续护盾
  - 随后朝目标发起短距离冲锋
  - 冲锋路径上被撞到的敌方英雄会被击飞到空中
  - 该技能的核心收益来自“抢先接敌并打断敌方前排节奏”，不额外承担主要爆发伤害职责
- 检测条件：
  - 小技能 CD 转好
  - 施法距离 `3` 内存在合法敌方英雄
- 当前关键参数：
  - 施法距离：`3`
  - 护盾值：`60`
  - 护盾持续时间：`4 秒`
  - 路径碰撞半径：`0.8`
  - 击飞持续时间：`1 秒`
  - 冲锋过程时长：`0.25 秒`
  - 冷却时间：`8 秒`

## 大招：Skyquake

- 技能类型：近身范围伤害 + 范围击飞
- 释放目标：以自己为圆心，立刻命中 `5` 范围内所有敌方英雄
- 效果：
  - 立即对范围内敌方英雄造成一次范围伤害
  - 同时附加短时击飞
  - 大招强调“贴身爆发开团”，要求格斗家已经压进敌阵，而不是远距离起手
- 检测条件：
  - 本局尚未释放过大招
  - 优先要求自身 `5` 范围内至少存在 `3` 名敌方英雄
  - 若开战 `35 秒` 后仍未满足，则阈值降低为至少 `2` 名敌方英雄
  - 若开战 `50 秒` 后仍未满足，则阈值降低为至少 `1` 名敌方英雄
- 当前关键参数：
  - 作用半径：`5`
  - 伤害倍率：基于攻击力 `1.8`
  - 击飞持续时间：`1 秒`
- 当前关键约束：
  - 主动释放型大招本局仅能释放 `1 次`

## 落地约束

- 这名英雄优先用统一技能系统、统一状态系统和统一位移能力表达，不应默认拆成整套战士专属脚本
- 小技能至少需要支持：
  - 对施法者自身施加护盾
  - 以目标为终点的短距离冲锋
  - 按冲锋路径判定命中的敌方单位并附加击飞
- 如果现有技能框架暂时表达不了，应该优先补“effect 级目标归属”与“线性冲锋路径命中”这两类通用能力，而不是长期把 `Skybreaker` 写成孤立特例

## 设计边界

- 格斗家的核心职责是近身开团和局部打断，不承担坦克式长期承伤职责
- 不应把格斗家做成高频、大范围、稳定覆盖全队的强控核心
- 不应把格斗家做成高机动切后排刺客
- 不应把格斗家的小技能扩成兼具高爆发、高护盾和长时间控制的全能技能
- 第一阶段下，格斗家优先验证“近战普攻 + 护盾冲锋 + 贴身范围击飞大招”这套战士模板
