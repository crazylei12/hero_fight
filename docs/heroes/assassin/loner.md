# Loner

## 基础信息

- 英文名：Loner
- 中文名：独行者
- 英雄 ID：`assassin_004_loner`
- 英雄分类：刺客
- 标签：近战、突进、收割、自我强化
- 核心定位：近战滚雪球刺客，主动拒绝来自其他友军的治疗和增益，依靠短时不可选中稳定切到当前最低血量敌人，并在每次击杀参与后持续抬高自己的攻击力与攻速、顺势回一口血
- 默认 AI 模板：`assassin_default`
- 当前定位补充：作为 `assassin_001_shadowstep`、`assassin_002_tidefin` 和 `assassin_003_butcher` 之后的第四个刺客扩展原型，不替代忍者的高频瞬移模板、不替代腐蚀虫的持续削弱模板，也不替代屠夫的阵型搅乱模板，主要验证“拒收友方正向效果 + 最低血量切入 + 击杀参与滚雪球被动大招”这条更极端的独狼刺客方向

## 普攻类型

- 类型：近战瞬时命中普攻
- 释放目标：按统一 `NearestEnemy` 目标策略选择最近的合法敌方英雄；若当前目标仍合法，则优先持续追击，不在多个目标之间无谓来回切换
- 效果：造成基于攻击力 `1.0` 倍率的单体伤害
- 检测条件：
  - 当前存在合法敌方目标
  - 目标进入攻击范围
  - 普攻间隔转好
- 当前关键参数：
  - 攻击距离：`1.35`
  - 普攻间隔：`0.83 秒`

## 小技能：Lone Pursuit（孤影追袭）

- 技能类型：短时 `不可选中` + 定点突进
- 释放目标：优先锁定全场当前 `绝对生命值` 最低的合法敌方英雄；若并列，则优先当前生命值比例更低者；若仍并列，则优先距离自己更远、通常更靠后的目标
- 效果：
  - 施法后立即进入一小段 `不可选中`
  - 在该窗口内高速冲向目标身边的合法近身落点
  - 到位后立即把该目标设为当前追击目标，并继续接续普攻
  - 该技能当前不额外附带独立伤害、护盾或硬控；核心收益是稳定抢到残血收割位
  - 这段 `不可选中` 不等于 `无敌`；独行者仍可被非指向范围效果、路径效果和持续区域效果命中
- 检测条件：
  - 小技能 CD 转好
  - 场上存在合法敌方目标
  - 能找到目标身边的合法落点
- 当前关键参数：
  - 施法距离：`全屏`，当前 `arena_stage01_flat` 可按 `56` 处理
  - 不可选中时长：`0.45 秒`
  - 冲刺时长：`0.45 秒`
  - 落点距离：目标身边 `0.9`
  - 冷却时间：`5.5 秒`

## 大招：Lone Instinct（孤战本能）

- 技能类型：被动型大招 / 常驻自我规则 + 击杀参与滚雪球
- 释放目标：自己；常驻生效，无主动施放动作
- 效果：
  - 独行者 `不接受来自其他友军的正向效果`
  - 默认包含：直接治疗、持续治疗、护盾、无敌、不可选中、分担伤害、正向属性 Buff、正向受治疗率修正，以及等价的范围正向效果
  - 因此其他友军的治疗普攻、治疗技能、增益技能和保护技能，都不应再把独行者当作合法目标
  - 这条约束只屏蔽“其他友军 -> 独行者”的正向效果；独行者自己的自我强化、自我治疗，以及敌方施加的负向效果不受影响
  - 每当独行者发生一次 `击杀参与`，包括击杀和助攻，立即获得 `1` 层永久持续到本局结束的自我强化
  - 每层提供一次 `AttackPowerModifier`
  - 每层攻击力提升 `+8%`
  - 每层同时提供一次 `AttackSpeedModifier`
  - 每层攻速提升 `+10%`
  - 触发时若独行者当前存活，则立即回复 `18%` 最大生命值
  - 上述层数在死亡后不清空，复活后继续保留
- 检测 / 生效条件：
  - 大招常驻生效，无需主动释放
  - `击杀参与` 默认沿用共享的击杀 / 助攻判定
  - 若独行者在死亡后因先前贡献获得助攻，仍可获得层数，但不追发延迟治疗
- 当前关键参数：
  - 每层攻击力提升：`+8%`
  - 每层攻速提升：`+10%`
  - 每次触发回复：`18%` 最大生命值
  - 最大层数：`5`
  - 层数持续时间：`至本局结束`
- 当前关键约束：
  - 这是 `被动型大招` 特例，不参与主动释放型大招的“一局一次施放”流程
  - 这名英雄的大招强度重点放在整局滚雪球与目标合法性约束，不依赖单次爆发按钮

## 落地约束

- 独行者当前定位为“第四个刺客扩展原型”，不替代现有三名刺客的既有模板；它的核心独特点必须稳定落在“拒收友方正向效果”上
- `Lone Pursuit` 应优先复用共享的 `Untargetable + reposition` 组合表达，不要新增“独行者冲刺”专属位移底层
- “当前绝对生命值最低的敌人”应优先沉淀为共享目标策略或统一 helper，不要只在独行者脚本里私下写一段孤立排序
- `Lone Instinct` 对友方正向效果的拒收，必须作用到友军普攻目标选择、技能目标选择、范围命中收集、直接命中合法性复检和 AI 治疗 / 增益决策；不要只在某个治疗技能上补特判
- `Lone Instinct` 的攻击力与攻速成长，应继续复用统一属性修正汇总入口，不要在普攻伤害公式或攻击计时器里私下累加隐藏变量
- 击杀参与回血应继续走统一治疗 / 回复入口，但来源应明确记为独行者自身；这样才能稳定复用日志、统计和受治疗率语义
- 日志和事件至少要能明确记录：友方正向效果因 `Lone Instinct` 被过滤、每次击杀参与获得的层数、当次回复量，以及当前总层数

## 设计边界

- 独行者的核心职责是切残血、滚雪球和自我续战，不承担前排承伤、团队保护或稳定团控职责
- 不应把独行者做成虽然文档写着“拒收友方正向效果”，但实战里仍能吃到队友治疗、护盾或 Buff 的伪独狼英雄
- 不应把 `Lone Pursuit` 扩成附带高额爆发、长时间不可选中或硬控链的全能切入技；它当前只负责抢收割位
- 不应把 `Lone Instinct` 扩成无限叠层、无上限攻速成长或击杀参与后瞬间回满血的失控被动
- 不应把“拒收友方正向效果”误解成免疫敌方负面效果、范围伤害或持续伤害；那会把英雄从独狼刺客改成规则特权怪
- 第一阶段下，独行者优先验证“近战普攻 + 最低血量突进小技能 + 击杀参与滚雪球被动大招 + 友方正向效果免疫”这套极端自给型刺客模板
