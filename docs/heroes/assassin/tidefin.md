# Tidefin

## 基础信息

- 英文名：Tidefin
- 中文名：腐蚀虫
- 英雄 ID：`assassin_002_tidefin`
- 英雄分类：刺客
- 标签：近战、突进、减益、收割
- 核心定位：近战削弱型刺客，通过普攻持续压低目标攻击力，再用短距跳跃扑向低血量敌人施加减防，并在一局一次的大招窗口里同步削弱一片敌人的攻防
- 默认 AI 模板：`assassin_default`
- 当前定位补充：作为 `assassin_001_shadowstep` 之后的第二个刺客原型，不替代忍者的高机动瞬移刺客模板，主要验证“普攻 on-hit 叠层 Debuff + 短距跳跃收割 + 范围双减益大招”这条刺客扩展方向

## 普攻类型

- 类型：近战瞬时命中普攻
- 释放目标：默认沿用刺客模板优先寻找敌方后排或脆皮；已经贴住目标时，优先持续追打当前更容易被收掉的低血量敌人
- 效果：
  - 造成基于攻击力 `1.0` 倍率的单体伤害
  - 命中后为目标附加一次 `AttackPowerModifier`
  - 单层减攻幅度为 `-5%`
  - 单次持续时间为 `3 秒`
  - 同目标可叠加至 `5 层`
  - 重复命中时刷新持续时间
- 检测条件：
  - 当前存在合法敌方目标
  - 目标进入攻击范围
  - 普攻间隔转好
- 当前关键参数：
  - 攻击距离：`1.3`
  - 普攻间隔：`0.90 秒`

## 小技能：Tidal Pounce

- 技能类型：短距跳跃切入 + 单体防御减益 Debuff
- 释放目标：施法距离 `4` 内当前累计造成伤害最高的敌方英雄；若并列则优先当前生命值更低者，若仍并列则优先当前距离更近者
- 效果：
  - 跳向目标身边的合法近身落点
  - 落地后立即为目标附加一次 `DefenseModifier`
  - 减防幅度为 `-20%`
  - 减防持续 `5 秒`
  - 该技能当前不额外附带独立伤害段，主要收益来自贴近高输出目标、压低其防御并为后续普攻和队友集火打开限制输出窗口
- 检测条件：
  - 小技能 CD 转好
  - `4` 范围内存在合法敌方目标
  - 能找到目标身边的合法落点
- 当前关键参数：
  - 施法距离：`4`
  - 落点距离：目标身边 `0.9`
  - 跳跃位移时长：`0.28 秒`
  - 跳跃抛物线峰值高度：`0.85`
  - 减防幅度：`-20%`
  - 减防持续时间：`5 秒`
  - 冷却时间：`5.5 秒`

## 大招：Ruin Tide

- 技能类型：自身范围双属性减益 Debuff
- 释放目标：自己；对自身 `4` 范围内的所有敌方英雄生效
- 效果：
  - 立即为范围内所有敌人附加一次 `AttackPowerModifier`
  - 同时附加一次 `DefenseModifier`
  - 两种减益幅度都为 `-20%`
  - 两种减益都持续 `5 秒`
  - 大招核心收益是同时压低敌方对拼能力和承伤能力，制造一段适合刺客收割与队友跟进的集体弱化窗口
- 检测条件：
  - 本局尚未释放过大招
  - 优先在自身 `4` 范围内至少存在 `2` 名敌人时释放
  - 若其中至少有 `1` 名敌人的生命值比例不高于 `50%`，则显著提高释放优先级
  - 若开战 `40 秒` 后仍未满足双目标窗口，则只要自身 `4` 范围内存在合法敌人即可释放
- 当前关键参数：
  - 生效半径：`4`
  - 攻击力降低：`-20%`
  - 防御力降低：`-20%`
  - 持续时间：`5 秒`
- 当前关键约束：
  - 主动释放型大招本局仅能释放 `1 次`

## 落地约束

- 腐蚀虫当前定位为“第二个刺客扩展原型”，不替代 `Shadowstep` 作为首个高机动瞬移刺客模板
- 普攻减攻必须直接复用统一的“普攻命中附带效果”配置能力，不应偷偷实现成腐蚀虫专属被动脚本
- 普攻附带的 `AttackPowerModifier` 应按统一状态系统表达为 `-5%`、`3 秒`、`maxStacks = 5`、重复命中刷新持续时间，不单独维护一套英雄私有层数计数器
- `Tidal Pounce` 应优先复用统一的短距 reposition + 直接附加状态效果组合，不应新增“腐蚀虫跳跃”专属移动底层
- `Tidal Pounce` 当前应通过共享抛物线位移参数来表达“跳跃”而不是闪烁：机制上接近短距冲锋，但表现上必须允许明显的离地高度与落地反馈
- `Tidal Pounce` 的“范围内造成伤害最高的敌人，若并列则选当前生命值更低者”应优先沉淀为可复用的目标策略或统一 helper，不要只在单英雄流程里硬编码
- `Ruin Tide` 应优先复用统一的范围 Debuff 技能结构，一次施法同时附加两个正式状态，而不是额外创建“腐蚀虫诅咒”专属底层
- 大招虽然不直接造成伤害，但仍应通过统一事件和日志明确记录命中人数、附加状态和持续时间，方便后续调试与调数

## 设计边界

- 腐蚀虫的核心职责是切近低血量目标并削弱敌方对拼能力，不承担前排承伤职责
- 不应把腐蚀虫做成带长时间硬控链或超高瞬间爆发的全能刺客
- 不应把腐蚀虫做成纯站在前排挂光环的 Debuff 工具人，必须保持刺客的贴身收割压力
- 不应把腐蚀虫扩成全屏切入、长时间不可选中或远程消耗型刺客
- 第一阶段下，腐蚀虫优先验证“近战普攻叠层减攻 + 锁高输出目标跳跃减防 + 一局一次范围攻防双削弱”这套削弱型刺客模板
