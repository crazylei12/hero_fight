# Shrinemaiden

## 基础信息

- 英文名：Shrinemaiden
- 中文名：巫女
- 英雄 ID：`support_004_shrinemaiden`
- 英雄分类：辅助
- 标签：远程、治疗、消耗、部署
- 核心定位：中后排双态节奏辅助，普攻在“对敌压血”和“对友补血”之间严格交替，用单体治疗联动范围反压维持前线换血，再用一局一次的图腾把这套双态普攻短时间内加速复制到战场上
- 默认 AI 模板：`support_default`
- 当前定位补充：作为 `support_001_sunpriest`、`support_002_windchime` 和 `support_003_monk` 之后的第四个辅助扩展原型，不替代圣职者的纯续航保人模板、不替代风语司铃的反切入保护模板，也不替代僧侣的前排群体续航模板，主要验证“固定轮转双态普攻 + 单体治疗联动范围伤害 + 部署型高速双态图腾大招”这条节奏型辅助方向

## 普攻类型

- 类型：双态远程投射物普攻 / 固定顺序轮转
- 释放目标：
  - 当前轮次为 `伤害弹` 时，按共享 `NearestEnemy` 目标策略选择最近的合法敌方英雄
  - 当前轮次为 `治疗弹` 时，优先选择当前生命值比例最低的合法友方英雄，包括自己；若当前没有受伤友军，则回退到最近友军，必要时可回退到自己作为施放锚点
  - 轮转顺序固定为 `伤害弹 -> 治疗弹 -> 伤害弹 -> 治疗弹`
  - 若当前轮次为 `伤害弹`，但全场已经没有任何合法敌方目标，而仍存在合法友方治疗目标，则允许本次出手显式回退为 `治疗弹`，避免巫女在敌方全灭等待复活期间卡在原地空等
- 效果：
  - `伤害弹` 对目标造成基于攻击力 `0.95` 倍率的单体伤害
  - `治疗弹` 对目标造成基于攻击力 `0.9` 倍率的单体治疗
  - 出生和复活后，普攻轮转默认重置为 `伤害弹`
  - 上述“敌方全空时回退到治疗弹”的行为只属于 `目标阵营完全缺失` 时的显式回退，不代表巫女平时会因为某个近处敌人短暂失效就随意打破双态轮转
  - 当前默认约束为：这次回退不会消耗或推进被跳过的 `伤害弹` 轮次；只要敌方重新出现合法目标，下一次正常普攻仍应从 `伤害弹` 开始
  - 两种普攻都走投射物，但必须提供可显式区分的表现标记：`attack_damage` 与 `attack_heal`
  - 这两个标记既要能区分巫女本体普攻，也应允许图腾复用，必要时可再叠加 `emitterType` 之类的来源标识，便于表现层分别挂接两套投射物和命中特效
- 检测条件：
  - 当前轮次存在合法目标
  - 目标进入攻击范围
  - 普攻间隔转好
- 当前关键参数：
  - 攻击距离：`8.2`
  - 普攻间隔：`1.05 秒`
  - 投射物速度：`14`
  - 出生 / 复活起手轮次：`伤害弹`

## 小技能：Prayer Bloom（祈愿绽华）

- 技能类型：单体治疗 + 目标周围范围伤害
- 释放目标：优先选择施法距离 `9.0` 内当前生命值比例最低、且自身 `3.6` 范围内至少存在 `1` 名敌方英雄的合法友军，包括自己；若没有满足联动伤害条件的友军，则回退到施法距离内生命值比例最低的合法友军
- 效果：
  - 立即为目标回复基于攻击力 `1.35` 倍率的生命值
  - 同时以该目标为中心，对半径 `3.6` 内所有敌方英雄造成基于攻击力 `1.0` 倍率的范围伤害
  - 这招的核心收益是“把被集火的友军抬一口，同时顺手压低贴在他身边换血的敌人血线”，不提供护盾、击飞或减益
- 检测条件：
  - 小技能 CD 转好
  - 施法距离 `9.0` 内存在合法友方目标
  - 优先在目标已受伤，或目标周围存在至少 `1` 名敌方英雄时释放
- 当前关键参数：
  - 施法距离：`9.0`
  - 治疗倍率：基于攻击力 `1.35`
  - 敌方伤害半径：`3.6`
  - 敌方伤害倍率：基于攻击力 `1.0`
  - 冷却时间：`8 秒`

## 大招：Twin Rite Totem（双仪图腾）

- 技能类型：部署型固定战斗代理体 / 高速双态投射
- 释放目标：优先在施法距离 `9.5` 内，选择当前生命值比例最低且自身 `4.5` 范围内至少有 `1` 名敌方英雄的友方单位当前位置召唤；若当前没有明显受压友军，则在施法距离内选择当前交战最密集的友军附近位置召唤
- 效果：
  - 在目标点召唤一座固定不移动的 `双仪图腾`
  - 图腾默认 `不可移动 + 不可选中 + 无敌`
  - 图腾持续 `6 秒`
  - 图腾会独立按 `伤害弹 -> 治疗弹` 的固定顺序轮转发射飞弹，逻辑与巫女本体普攻同源，但频率更快
  - 图腾的 `伤害弹` 会优先攻击自己 `8.5` 范围内最近的合法敌方英雄，对目标造成基于攻击力 `0.8` 倍率的单体伤害
  - 图腾的 `治疗弹` 会优先治疗自己 `8.5` 范围内生命值比例最低的合法友军，包括巫女本人；若当前没有受伤友军，则回退到最近友军，必要时可回退到巫女本人作为施放锚点，并保持轮次不变
  - 若图腾当前轮次为 `伤害弹`，但自己范围内已不存在任何合法敌方目标，而仍存在合法友方治疗目标，也允许本次出手显式回退为 `治疗弹`，避免图腾在敌方暂时全灭时进入纯等待
  - 图腾生成后第一发默认为 `伤害弹`
  - 图腾与本体保持同一约束：这类回退不会推进被跳过的 `伤害弹` 轮次；敌方重新进入合法范围后，下一次正常发射仍从 `伤害弹` 开始
  - 图腾飞弹同样必须显式区分 `attack_damage` 与 `attack_heal` 两种表现标记；如果表现层还需要区分“本体飞弹”和“图腾飞弹”，应继续复用共享的来源标识，而不是为巫女单独发明第三套攻击类型
  - 大招核心收益是用一段时间内更高频的双态普攻，同时补血和压血，制造一小段己方前线更稳、敌方前线更难换血的窗口
- 检测条件：
  - 本局尚未释放过大招
  - 优先要求施法距离 `9.5` 内存在至少 `1` 名生命值比例不高于 `70%`、且自身 `4.5` 范围内至少有 `1` 名敌方英雄的友军
  - 若当前没有明显低血受压友军，则只要施法距离内能找到图腾 `8.5` 范围同时覆盖至少 `2` 名友军和 `2` 名敌方英雄的交战点即可释放
  - 若开战 `45 秒` 后仍未满足上述理想窗口，则门槛放宽为图腾 `8.5` 范围内至少能稳定覆盖 `2` 个合法目标
- 当前关键参数：
  - 施法距离：`9.5`
  - 图腾持续时间：`6 秒`
  - 图腾攻击距离：`8.5`
  - 图腾发射间隔：`0.5 秒`
  - 图腾投射物速度：`16`
  - 图腾伤害弹倍率：基于攻击力 `0.8`
  - 图腾治疗弹倍率：基于攻击力 `0.8`
  - 图腾起手轮次：`伤害弹`
- 当前关键约束：
  - 主动释放型大招本局仅能释放 `1 次`

## 落地约束

- 巫女当前定位为“第四个辅助扩展原型”，不替代 `support_001_sunpriest` 的纯续航保人模板，也不替代 `support_002_windchime` 的反切入保护模板，更不替代 `support_003_monk` 的贴前排群体续航模板
- 巫女本体普攻和图腾飞弹都应优先复用共享的“固定顺序轮转型普攻”能力，不要把“伤害弹 / 治疗弹切换”散写成巫女专属的 if 分支或表现层计数器
- 当前轮转序号应分别挂在巫女本体和图腾各自的运行时状态上；两者都允许按配置在出生、复活或图腾创建时重置到 `伤害弹` 起手，不要把轮次状态回写到静态配置
- 对巫女而言，应额外显式支持“敌方目标阵营完全缺失时，伤害弹回退到治疗弹继续出手”这条共享回退规则；它的目的仅是避免敌方全灭窗口内原地空转，不是把巫女改成会随时打破轮转的自由切换型辅助
- `attack_damage` 与 `attack_heal` 这两个表现标记必须作为正式逻辑输出继续传到投射物和命中事件；如果后续要给两种普攻做完全不同的特效，应直接复用这两个标记，不要再靠 `heroId` 或动画状态机猜当前是哪一发
- `Prayer Bloom` 应优先复用共享的“单体治疗 + 目标周围敌方范围伤害”技能表达，不要为巫女单独发明常驻祈福圈、额外连线检测或伪持续区域系统
- `Twin Rite Totem` 必须复用统一的 `部署型固定战斗代理体` 语义，不要把图腾做成第二个完整 `RuntimeHero`、独立计分单位或拥有完整移动 / AI 的伪英雄
- 图腾造成的伤害、治疗、日志和表现事件，默认仍应挂回巫女及其大招来源；至少要能明确记录：图腾召唤时机、当前轮次、每次伤害弹或治疗弹命中的目标，以及图腾结束时间

## 设计边界

- 巫女的核心职责是用严格交替的双态普攻做持续压血与补血，不承担高爆发输出核心或纯主治疗核心职责
- 不应因为当前没有受伤友军，就把 `治疗弹` 偷偷改成额外伤害弹；“固定轮转”本身就是这名英雄的核心识别点
- 但如果当前已经不存在任何合法敌方目标，而友方仍有可治疗对象，则不应让巫女或图腾卡在等待 `伤害弹`；这种极端窗口下应继续治疗
- 不应把巫女的小技能扩成同时附带护盾、减伤、击飞和减益的全能保护按钮；它当前只负责单点抬血并反压贴脸敌人
- 不应把图腾做成可移动、可同时存在多座、带强控或能独自扛线的第二角色；它应只是短时存在的节奏放大器
- 不应把巫女做成依赖图腾打满持续伤害的炮台型输出辅助；图腾的主要价值仍是同时补血和压血，而不是替代射手或法师的持续输出职责
- 第一阶段下，巫女优先验证“远程双态轮转普攻 + 单体治疗联动范围伤害小技能 + 部署型高速双态图腾大招”这套节奏型辅助模板
