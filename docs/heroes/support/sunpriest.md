# Sunpriest

## 基础信息

- 英文名：Sunpriest
- 中文名：圣职者
- 英雄 ID：`support_001_sunpriest`
- 英雄分类：辅助
- 标签：远程、治疗、增益
- 核心定位：中后排单点保人辅助，依靠友方投射物普攻稳定抬血，再用单体保护和范围续航维持团战
- 默认 AI 模板：`support_default`

## 普攻类型

- 类型：远程投射物治疗普攻
- 释放目标：优先当前最低血量友军，包括自己；当没有受伤友军时，会把最近友军当成站位锚点，不对敌人造成伤害
- 效果：发射一枚治疗投射物，对命中的友军造成基于攻击力 `1.0` 倍率的单体治疗
- 检测条件：
  - 当前存在合法友方目标
  - 目标进入攻击范围
  - 普攻间隔转好
- 当前关键参数：
  - 攻击距离：`7.8`
  - 普攻间隔：`1.11 秒`
  - 投射物速度：`14`

## 小技能：Radiant Heal

- 技能类型：单体治疗 + 护盾
- 释放目标：施法范围内血量最低的友方英雄，包括自己
- 效果：
  - 立即治疗目标
  - 同时附加一个持续时间有限的护盾
  - 当前参数下更偏向“快速抬血并防止被补刀”的保人技能
- 检测条件：
  - 小技能 CD 转好
  - 施法距离 `9` 内存在合法友方目标
- 当前关键参数：
  - 施法距离：`9`
  - 直接治疗倍率：基于攻击力 `1.35`
  - 护盾值：`45`
  - 护盾持续时间：`4 秒`
  - 冷却时间：`7 秒`

## 大招：Sun Blessing

- 技能类型：持续范围治疗
- 释放目标：以施法范围内血量最低的友军当前位置为中心释放
- 效果：
  - 在目标点生成一个固定不跟随的祝福区域
  - 对区域内所有友军持续治疗
  - 每跳治疗为基于攻击力 `0.65` 倍率的范围治疗
  - 区域半径为 `7.5`
  - 持续 `5 秒`
  - 每 `1 秒` 脉冲一次
  - 释放后即与圣职者本体解耦，后续移动、普攻、释放别的技能或死亡，都不会提前结束该区域
- 检测条件：
  - 本局尚未释放过大招
  - 优先要求施法距离 `9` 内的目标点半径 `7.5` 内至少存在 `2` 名友军
  - 且其中至少 `1` 名友军生命值比例不高于 `55%`
  - 若开战 `30 秒` 后仍未满足，则低血量阈值放宽到 `70%`
  - 若开战 `45 秒` 后仍未满足，则低血量阈值放宽到 `85%`
- 当前关键约束：
  - 主动释放型大招本局仅能释放 `1 次`

## 设计边界

- 圣职者的核心职责是续航和保人，不承担持续输出或爆发收割职责
- 不应把圣职者做成靠普攻点杀敌人的远程输出位
- 不应把圣职者做成大范围硬控核心
- 第一阶段下，圣职者优先验证“友方投射物普攻 + 单体治疗护盾 + 持续范围治疗大招”这套辅助模板
