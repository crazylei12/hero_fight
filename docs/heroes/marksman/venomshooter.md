# Venomshooter

## 基础信息

- 英文名：Venomshooter
- 中文名：毒箭手
- 英雄 ID：`marksman_003_venomshooter`
- 英雄分类：射手
- 标签：远程、持续输出、DOT、范围消耗、连锁收割
- 核心定位：后排持续压血型射手，依靠普攻稳定挂毒，用范围施毒扩大敌方中毒覆盖率，再以一局一次的大招把既有毒层转化为团战收割与连锁崩盘
- 默认 AI 模板：`marksman_default`
- 当前定位补充：作为 `marksman_002_rifleman` 之后的第三个射手扩展原型，不替代 `Longshot` 的“拉扯 + 连射收割”模板，也不替代 `Rifleman` 的“慢速高压制 + 范围爆发”模板，主要验证“普攻 on-hit 挂毒 + 范围施毒扩散 + 按既有毒层收割并触发连锁爆炸”这条 DOT 流射手方向

## 当前数值倾向

- 直接普攻压制力应低于 `Longshot`，主要伤害差异来自持续毒层而不是单发爆击
- 单体连续站桩输出时间越长，毒箭手的威胁越高；因此普攻节奏可以中速，但单发不应太重
- 生存性应略低于 `Longshot`、高于 `Rifleman`，体现“没有强自保，但也不是纯脆皮炮台”
- 暴击收益不是这名英雄的主要强点，暴击率和暴击伤害都不应高于现有两名射手

## 当前实现基线

- 最大生命：`295`
- 攻击力：`32`
- 防御力：`8`
- 攻速：`1.111`，约等于 `0.90 秒 / 次`
- 移速：`3.8`
- 暴击率：`14%`
- 暴击伤害：`1.75`
- 攻击距离：`9`

## 普攻类型

- 类型：远程投射物普攻
- 释放目标：默认沿用射手模板维持安全输出位置，持续攻击当前合法目标；被近身威胁压迫时仍以短时后撤为主，不依赖位移或强控脱身
- 效果：
  - 发射毒箭，对命中的敌方目标造成一次正常普攻伤害
  - 命中后附加一层以“中毒”为主题表达的 `DamageOverTime`
  - 普攻是毒箭手最稳定的挂毒来源，职责是持续维持单体毒层，而不是在普攻上直接塞入额外爆发段
- 检测条件：
  - 当前存在合法敌方目标
  - 目标进入攻击范围
  - 普攻间隔转好
- 当前关键参数：
  - 普攻伤害倍率：基于攻击力 `0.6`
  - 攻击距离：`9`
  - 普攻间隔：`0.90 秒`
  - 投射物速度：`15`
  - 单次附加层数：`1`
  - 中毒单层持续时间：`4 秒`
  - 中毒 tick 间隔：`1 秒`
  - 毒箭手自身这套毒默认共用 `1` 组 `5` 层上限的本地叠层池
  - 中毒单层每跳伤害：建议按施加时攻击力 `0.3` 倍换算；若按当前 `32` 攻击力基线估算，对 `10` 防御标准脆皮约为 `8.7 / 跳 / 层`，满 `5` 层约为 `43.6 / 秒`
  - 重复命中：对毒箭手自己的同源毒池刷新持续时间，并在未到上限时继续叠层
  - 未来其他来源施加的 `Poison / 中毒` 主题 DOT 不与这组 `5` 层上限共享，但会在后续主题读取中一起被统计

## 小技能：毒雾瓶

- 技能类型：范围施毒 / 群体消耗
- 释放目标：优先朝施法距离内敌方较密集的位置投掷；若当前攻击目标附近已经形成小规模抱团，也可直接朝该目标所在区域施放
- 效果：
  - 朝目标区域投掷毒雾瓶
  - 毒瓶破裂后，对区域内所有敌方英雄附加中毒
  - 该技能核心职责是把普攻形成的单体挂毒扩展成群体挂毒，为后续大招创造“多人已带毒”的启动条件
  - 小技能本身不应承担主爆发职责，重点仍是扩大覆盖面和压低整体血线
- 检测条件：
  - 小技能 CD 转好
  - 施法距离内能找到合法敌方区域锚点
  - 当前至少能命中 `1` 名敌方英雄
- 当前关键参数：
  - 施法距离：`9.2`
  - 作用半径：`3.6`
  - 投掷飞行时间：`0.35 秒`
  - 冷却时间：`6 秒`
  - 单个目标初始附加层数：`3`
  - 附加的中毒与普攻附毒共用同一 `Poison / 中毒` 主题键；对毒箭手自身默认也共用同一来源内的 `5` 层叠层池
  - 中毒持续时间：`4 秒`
  - 中毒 tick 间隔：`1 秒`
  - 同一来源内的中毒最大叠层数：`5`

## 大招：毒爆收割

- 技能类型：按既有毒层结算的群体收割 / 击杀后范围连锁
- 释放目标：自己；只对当前已经带毒的敌方英雄进行结算
- 效果：
  - 对所有当前带毒的敌方英雄造成一次与其当前毒层直接相关的额外伤害
  - 大招本体的收益来自“把前期积累的毒层兑现成一轮同步爆发”，而不是重新铺一层新的全场中毒
  - 若某个目标因这次大招结算死亡，则以其死亡位置触发一次毒性爆炸
  - 毒性爆炸会对周围敌方英雄造成范围伤害，并附加或扩散中毒，从而形成进一步的连锁压血与收割
- 检测条件：
  - 本局尚未释放过大招
  - 优先要求场上至少有 `3` 名敌方英雄处于中毒状态
  - 若当前已有 `2` 名带毒敌方英雄生命值比例不高于 `50%`，则也允许提前释放
  - 若开战 `30 秒` 后仍未满足，则阈值降低为至少 `2` 名带毒敌方英雄
  - 若开战 `45 秒` 后仍未满足，则只要场上仍存在至少 `1` 名合法带毒目标，也允许提前开大尝试收口
- 当前关键参数：
  - 大招本体基础伤害：`0`
  - 每层毒层额外伤害：基于攻击力 `1.5`
  - 读取层数范围：统计目标身上所有 `Poison / 中毒` 主题 DOT 的总层数，不限来源；但不混算 `Burn / Bleed / Curse` 等其他主题 DOT
  - 不同来源或不同技能的毒层继续各自管理持续时间和层数上限；只在大招读取阶段做统一汇总
  - 毒层读取后默认不消耗已有毒层
  - 按当前 `32` 攻击力基线估算，对 `10` 防御标准脆皮：`5` 层毒的大招本体约造成 `218` 点伤害；这份数值的目标不是“从零直接蒸发所有满血目标”，而是让 `4~5` 层已铺毒目标稳定进入击杀线，符合一局一次大招的收割强度
  - 尸爆作用半径：`3.4`
  - 尸爆伤害：基于攻击力 `1.5`
  - 尸爆附加层数：`2`
  - 尸爆附加的中毒持续时间：`4 秒`
  - 尸爆附加的中毒 tick 间隔：`1 秒`
  - 尸爆触发范围：大招本体击杀和尸爆击杀都可继续触发；但同一单位在同一次大招结算链中最多触发 `1` 次尸爆
- 当前关键约束：
  - 主动释放型大招本局仅能释放 `1 次`

## 落地约束

- 毒箭手当前定位为“第三个射手扩展原型”，不替代 `Longshot` 作为首个拉扯型射手模板，也不替代 `Rifleman` 的单点压制模板
- 普攻附毒必须直接复用统一的“普攻命中附带效果”配置能力，不应偷偷实现成毒箭手专属被动脚本
- “中毒”优先作为共享 `DamageOverTime` 的主题化表达处理，不新增毒箭手私有的底层 `Poison` 状态类型
- `毒雾瓶` 应优先复用统一的范围技能结构，表达为“命中区域内附加正式状态”或等价的共享区域载荷，不要单独新增“毒瓶专属结算底层”
- `毒爆收割` 的伤害计算应优先复用共享的“按目标已有正式状态层数结算收益”能力，不要在大招脚本里额外维护一套只给这名英雄使用的毒层缓存
- `毒爆收割` 读取的应是目标身上所有 `Poison / 中毒` 主题状态的总层数，而不是只读取毒箭手自己施加的那一份毒
- 大招默认应是“读取毒层但不消耗毒层”；如果后续决定要在结算后清空或折损毒层，必须先更新共享规则和本文档
- 尸爆应优先复用共享的“目标因指定技能或效果死亡后，在死亡位置触发范围效果”能力，不要求先实现可持续存在的尸体对象
- 尸爆产生的范围伤害与附毒扩散，仍应继续复用统一的范围伤害与范围附状态载荷，不要把整段连锁逻辑写成毒箭手专属的多层事件分支

## 设计边界

- 毒箭手的核心职责是持续挂毒、群体压血和条件收割，不承担高机动射手职责
- 不应把毒箭手做成高爆发秒杀型射手或依赖暴击瞬间决胜的传统站桩射手
- 不应把毒箭手做成强控制核心；它可以通过持续伤害逼迫敌方血线，但不应靠长时间硬控主导团战
- 不应把大招扩成“无论目标是否铺毒都稳定打满”的万能爆发按钮；它必须明显奖励先铺毒、后收割的节奏
- 第一阶段下，毒箭手优先验证“远程投射物普攻挂 DOT + 范围施毒扩面 + 一局一次按状态层数收割并触发死亡位置爆炸”这套 DOT 流射手模板
