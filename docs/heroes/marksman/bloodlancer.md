# Bloodlancer

## 基础信息

- 英文名：Bloodlancer
- 中文名：神灵武士
- 英雄 ID：`marksman_006_bloodlancer`
- 英雄分类：射手
- 标签：远程、持续输出、生命消耗、吸血、高风险
- 核心定位：中后排风险换伤害型射手，依靠燃血长矛主动压低自身生命值来换取更高普攻伤害，再通过随生命降低而增强的自我回复维持持续作战；在有辅助保护时收益很高，但被爆发或控制命中时仍会快速死亡
- 默认 AI 模板：`marksman_default`
- 当前定位补充：作为 `marksman_005_sniper` 之后的第六个射手扩展原型，不替代 `Longshot` 的拉扯收割、`Rifleman` 的笨重火力、`Venomshooter` 的 DOT 收割、`Boomeranger` 的短射程 AOE，也不替代 `Sniper` 的超长射程反后排；神灵武士主要验证“生命消耗 + 普攻追加伤害 + 缺血回复 + 一局一次自我强化窗口”这条高风险持续输出射手方向

## 当前数值倾向

- 基础生命应高于纯脆皮射手，但不能接近坦克；他靠主动压低血线换收益，而不是靠高防御硬吃伤害
- 基础防御应偏低到中低，保证大招期间仍然会被敌方爆发、刺客切入和集火直接击杀
- 基础攻击距离应短于 `Sniper`、不高于常规射手长射程档，避免在安全距离外无成本燃血输出
- 基础暴击收益不应过高；主要输出差异来自燃血长矛和大招强化，而不是高暴击秒人
- 生存强度高度依赖“能否持续命中并吸回来”；被眩晕、击飞、不可攻击或目标断档时，应明显暴露风险

## 建议数值基线

- 最大生命：`340`
- 攻击力：`31`
- 防御力：`7`
- 攻速：`1.0`，约等于 `1.00 秒 / 次`
- 移速：`3.6`
- 暴击率：`10%`
- 暴击伤害：`1.6`
- 攻击距离：`8.6`

## 普攻类型

- 类型：远程投射物普攻
- 释放目标：默认沿用射手模板维持中后排输出位置，持续攻击最近的可攻击敌人；被近身威胁压迫时仍可短时后撤，但不会获得位移、击退或硬控来自保
- 效果：
  - 投掷长矛，对命中的敌方目标造成一次正常远程普攻伤害
  - 若小技能 `Bloodburn Spear` 生效，则本次普攻命中后还会触发生命消耗、追加伤害和自我回复
  - 普攻本身不带穿透、弹射、范围伤害或后排点名
- 检测条件：
  - 当前存在合法敌方目标
  - 目标进入攻击范围
  - 普攻间隔转好
- 当前关键参数：
  - 基础普攻伤害倍率：基于攻击力 `0.85`
  - 攻击距离：`8.6`
  - 普攻间隔：`1.00 秒`
  - 投射物速度：`15`

## 小技能：Bloodburn Spear

- 技能类型：被动型小技能 / 普攻命中生命消耗 / 追加伤害 / 自我回复
- 触发目标：自身普攻命中的当前敌方目标
- 效果：
  - 神灵武士的每次普攻命中合法敌方英雄后，会先消耗自身一部分当前生命值
  - 生命消耗后，对本次命中目标追加一段基于攻击力的额外伤害
  - 同时神灵武士获得一次自我回复；当前生命值比例越低，这次回复越高
  - 该回复代表持续作战中的吸血收益，但不读取“本次实际造成伤害”的结果，不做成全局生命偷取属性
  - 这套机制的重点是让神灵武士主动压低血线并在持续命中中吸回来，而不是让他无成本获得常驻高伤害
- 检测条件：
  - 普攻成功命中合法敌方英雄
  - 神灵武士当前存活
  - 本次命中不是同一次普攻链额外复制出的伪命中
- 当前关键参数：
  - 生命消耗：自身当前生命值的 `4%`
  - 生命消耗下限：不会直接把自身生命值降到 `1` 以下
  - 额外伤害：基于攻击力 `0.45`
  - 自我回复：基于攻击力 `0.18 + 缺失生命值比例 * 0.42`
  - 回复结算：每次普攻命中后立即结算一次直接治疗，受目标侧 `HealTakenModifier` 等统一治疗修正规则影响
- 当前关键约束：
  - 这是唯一的小技能位，占用正式小技能，不额外新增独立被动栏
  - 生命消耗不是敌方造成的伤害，不应产生敌方击杀、助攻或受伤贡献
  - 生命消耗不应被自身护盾、防御、减伤、无敌或分担伤害抵消；它是技能代价，不是普通伤害事件
  - 追加伤害不单独暴击，不触发额外普攻链，也不再次触发燃血长矛

## 大招：Blood Oath

- 技能类型：主动型大招 / 自我生命消耗 / 短时攻速强化 / 普攻燃血强化
- 释放目标：自己
- 效果：
  - 神灵武士立刻消耗自身一部分当前生命值，进入短暂血誓状态
  - 血誓持续期间，神灵武士的攻击速度大幅提高
  - 血誓持续期间，`Bloodburn Spear` 的额外伤害提高，自我回复也提高
  - 血誓不会提供减伤、护盾、无敌、不可选中、免控或死亡防止
  - 若神灵武士在血誓期间被眩晕、击飞、嘲讽导致无法正常攻击，血誓持续时间仍然继续流逝
  - 若神灵武士在血誓期间死亡，血誓状态立即结束；复活后不继承血誓状态，但本局大招仍视为已释放
  - 该大招的强度来自“短时间内更敢打、更能吸回来”，不是一次性爆发或斩杀按钮
- 检测条件：
  - 本局尚未释放过大招
  - 当前至少存在 `1` 名合法敌方目标
  - 优先在自身生命值比例处于 `45% ~ 85%` 且当前攻击距离内存在合法目标时释放
  - 若自身正在获得友方护盾、持续治疗或分担伤害等正式保护收益，允许在生命值比例不低于 `30%` 时释放
  - 若自身生命值比例低于 `25%`，默认不主动开启，避免把大招变成自杀按钮；除非后续大招决策模板明确支持“有强保护时搏命开启”
  - 若开战 `42 秒` 后仍未释放，只要自身生命值比例不低于 `35%` 且场上存在合法敌方目标，即可释放争取最后输出窗口
- 当前关键参数：
  - 初始生命消耗：自身当前生命值的 `30%`
  - 生命消耗下限：不会直接把自身生命值降到 `1` 以下
  - 持续时间：`6 秒`
  - 攻速修正：`+75%`
  - 大招期间燃血长矛额外伤害：由基于攻击力 `0.45` 提高到 `0.80`
  - 大招期间自我回复：由基于攻击力 `0.18 + 缺失生命值比例 * 0.42` 提高到 `0.32 + 缺失生命值比例 * 0.65`
- 当前关键约束：
  - 主动释放型大招本局仅能释放 `1 次`
  - 大招期间不改变攻击距离，不让神灵武士获得全屏输出能力
  - 大招期间不改变索敌规则，不让普攻绕过前排直接点名后排

## 落地约束

- 神灵武士当前定位是“生命消耗型射手”，不应把核心收益写成战士、刺客或坦克式特例逻辑
- `Bloodburn Spear` 是被动型小技能，应作为正式小技能配置或共享监听配置表达，不要额外新增第四个被动技能槽
- 普攻命中的生命消耗、追加伤害和自我回复应优先落在共享的普攻 on-hit / 被动监听能力上，不应写成 `marksman_006_bloodlancer` 私有普攻脚本
- 生命消耗应作为“自身技能代价”进入统一事件和日志，至少能在调试日志中区分：
  - 自身生命消耗
  - 对敌方追加伤害
  - 自身回复
- 自我回复必须复用统一治疗结算和日志入口，不能在普攻流程里直接私下改当前生命值
- 大招的攻速提高应优先通过正式 `AttackSpeedModifier` 或等价运行时动作覆写表达；大招期间的燃血强化应通过明确的临时参数覆写表达
- 大招不应附带 `Invulnerable`、`Untargetable`、`DamageShare`、`DeathPrevent` 或任何减伤类隐藏效果
- 被硬控时，大招不提供免控，也不暂停持续时间；这点必须在实现和日志中保持一致，避免风险窗口被偷偷抹掉
- 死亡会清空大招状态；复活后不继承大招期间的攻速、伤害或回复强化

## 设计边界

- 神灵武士的核心职责是高风险远程持续输出，不承担前排承伤、团队保护或强控制职责
- 不应把神灵武士做成位移跳脸英雄；他可以通过燃血进入搏命输出状态，但不能混成刺客
- 不应给神灵武士添加无敌、不可选中、免控、高额减伤或稳定死亡防止，否则生命消耗的风险感会消失
- 不应把大招做成一次性斩杀、全屏点名或高爆发炮击；它必须保持为持续普攻强化窗口
- 不应让生命消耗被护盾或减伤抵消，也不应让自我回复绕过统一治疗修正；否则风险和反制都会变得不可读
- 不应让神灵武士在没有持续命中目标时仍稳定回血；他的续航必须依赖普攻命中
- 第一阶段下，神灵武士优先验证“远程投射物普攻 + 生命消耗追加伤害 + 缺血自我回复 + 一局一次高风险攻速强化”这套生命换伤害射手模板
