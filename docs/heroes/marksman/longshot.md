# Longshot

## 基础信息

- 英文名：Longshot
- 中文名：迅捷射手
- 英雄 ID：`marksman_001_longshot`
- 英雄分类：射手
- 标签：远程、持续输出、拉扯、收割
- 核心定位：后排持续输出射手，依靠高速投射物普攻稳定压血，用单点击退拉开贴身威胁，再用一局一次的连射大招快速收割残局
- 默认 AI 模板：`marksman_default`

## 普攻类型

- 类型：远程投射物普攻
- 释放目标：当前索敌目标；射手默认优先维持安全输出位置，持续攻击最近的可攻击敌人，被近身压迫时可先短时后撤再继续输出
- 效果：发射箭矢，对命中的敌方目标造成基于攻击力 `1.0` 倍率的单体伤害
- 检测条件：
  - 当前存在合法敌方目标
  - 目标进入攻击范围
  - 普攻间隔转好
- 当前关键参数：
  - 攻击距离：`6`
  - 普攻间隔：`0.74 秒`
  - 投射物速度：`16`

## 小技能：Heavy Shot

- 技能类型：单体伤害 + 击退
- 释放目标：当前攻击目标；若当前目标不合法，则退回最近的合法敌方英雄
- 效果：
  - 朝目标射出一发强力箭矢
  - 对目标造成基于攻击力 `1.7` 倍率的单体伤害
  - 同时对目标施加一次沿远离施法者方向的水平击退
  - 该技能核心价值是“打断敌方贴脸节奏并重新拉开输出距离”，不承担硬控职责
- 检测条件：
  - 小技能 CD 转好
  - 施法距离 `6` 内存在合法敌方目标
- 当前关键参数：
  - 施法距离：`6`
  - 伤害倍率：基于攻击力 `1.7`
  - 击退距离：`2.4`
  - 击退持续时间：`0.25 秒`
  - 冷却时间：`6 秒`

## 大招：Rapid Barrage

- 技能类型：连续单体普攻序列
- 释放目标：优先以当前攻击目标作为起始目标；若当前目标不合法，但场上仍存在合法敌方目标，则在全屏合法敌方单位中选择一个起始目标
- 效果：
  - 进入短时站桩连射状态，连续快速发射 `20` 次视为普通攻击的射击
  - 连射期间，迅捷射手的普攻射程会被临时覆写为 `全屏`
  - 当前第一阶段地图 `arena_stage01_flat` 的正式尺寸为 `48 x 27` 世界单位，因此这条“全屏”语义在实现上应至少覆盖整张地图；当前可按不低于 `56` 的临时射程处理
  - 每次射击都沿用普攻的投射物、暴击和单体命中规则，只覆写射程，不额外改写普攻伤害倍率
  - 当前目标死亡、不可选中或失去合法性后，会立即在全屏合法敌方单位中切换到下一个目标继续剩余射击
  - 大招不附带位移、无敌或霸体；若连射期间被眩晕、击飞或其他会打断攻击的效果命中，剩余射击直接取消
- 检测条件：
  - 本局尚未释放过大招
  - 当前至少存在 `1` 名合法敌方目标，不要求先进入普通攻击范围
  - 优先在起始目标生命值比例不高于 `65%` 时释放
  - 若起始目标附近 `6` 范围内至少存在 `2` 名敌方英雄，也可以提前释放以争取连续收割
  - 若开战 `40 秒` 后仍未满足，则只要当前存在合法目标即可释放
- 当前关键参数：
  - 总射击次数：`20`
  - 连射间隔：`0.25 秒`
  - 大招期间临时普攻射程：`全屏`，当前 `arena_stage01_flat` 可按 `56` 处理
  - 单次射击：沿用当前普攻伤害倍率和暴击规则
- 当前关键约束：
  - 主动释放型大招本局仅能释放 `1 次`

## 落地约束

- 当前工程中的 `marksman_001_longshot` 仍是占位实现；后续应直接沿用这个稳定英雄 ID，把占位技能方案替换为本文档定义的 `Heavy Shot / Rapid Barrage`
- 小技能的击退必须复用统一 `ApplyForcedMovement` 能力表达，不应把“击退”重新混写成 `KnockUp`
- `Rapid Barrage` 需要建立在一套通用的“连续动作序列”能力上，而不应只做成射手专属的“连续普攻特例”
- 这套通用序列能力至少要支持：
  - 保存剩余执行次数或剩余执行时长
  - 按固定间隔连续执行动作
  - 动作载荷既可以是 `普攻`，也可以是“重复释放同一个技能”或后续别的标准动作
  - 序列期间允许临时覆写动作参数，例如射程
  - 当前目标失效时按配置刷新目标
  - 被硬控或其他明确的中断条件打断时取消剩余序列
- `Rapid Barrage` 只是这套通用序列能力的第一个落地样例：当前序列载荷是“重复执行 20 次普攻”
- 如果现有技能模板暂时不足以表达这套逻辑，应优先补共享的序列执行入口，而不是把 `Longshot` 写成射手专属的大段特例脚本

## 设计边界

- 迅捷射手的核心职责是远程持续输出和残局收割，不承担法师式范围爆发职责
- 不应把迅捷射手做成高机动突进英雄或远程刺客
- 不应把迅捷射手做成依赖大范围硬控吃饭的控制核心
- 大招期间虽然拥有全屏射程，但仍必须遵守目标合法性校验，不应扩成无视 `不可选中`、死亡或其他失效条件的万能终结技
- 第一阶段下，迅捷射手优先验证“远程投射物普攻 + 单点击退拉扯 + 连续普攻大招”这套射手模板
