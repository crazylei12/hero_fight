# Boomeranger

## 基础信息

- 英文名：Boomeranger
- 中文名：轮子妈
- 英雄 ID：`marksman_004_boomeranger`
- 英雄分类：射手
- 标签：远程、持续输出、AOE、路径伤害、短手
- 核心定位：中后排范围压血型射手，依靠更短的攻击距离换取稳定的弹射普攻，用往返飞轮切穿局部战线，再用贴身旋轮大招惩罚前压和缠斗
- 默认 AI 模板：`marksman_default`
- 当前定位补充：作为 `marksman_003_venomshooter` 之后的第四个射手扩展原型，不替代 `Longshot` 的长射程拉扯模板，也不替代 `Rifleman` 的单点压制模板，更不替代 `Venomshooter` 的 DOT 铺场模板；主要验证“短射程弹射普攻 + 固定方向往返路径技能 + 跟随施法者的近身持续大招”这条 AOE 射手方向

## 当前数值倾向

- 基础攻击距离应明显短于现有三名射手，作为换取群体压血能力的主要代价
- 单目标持续输出不应超过 `Longshot` 或 `Rifleman` 的理想单点压制；优势只在敌方抱团、前排并肩或近战贴上来时快速放大
- 生存性可以略高于 `Rifleman`、接近 `Venomshooter`，但不应靠高坦度站进人群里硬吃集火
- 移速可略高于笨重射手，帮助她用更短攻击距离找弹射角度和大招贴身窗口，但不应因此变成高机动走 A 核心

## 当前实现基线

- 最大生命：`300`
- 攻击力：`31`
- 防御力：`8`
- 攻速：`1.053`，约等于 `0.95 秒 / 次`
- 移速：`3.9`
- 暴击率：`12%`
- 暴击伤害：`1.7`
- 攻击距离：`8.2`

## 普攻类型

- 类型：远程投射物普攻 + 连锁弹射
- 释放目标：默认沿用射手模板维持中后排输出位置，持续攻击最近的可攻击敌人；由于攻击距离更短，她会比其他射手更频繁地在前后排交界处停手，不应再额外给长射程保底
- 效果：
  - 向主目标掷出飞轮，对命中的首个敌方目标造成一次正常普攻伤害
  - 首次命中后，飞轮会继续向附近未被本次普攻链命中过的敌方英雄弹射，优先寻找上一跳命中点附近的最近目标
  - 每次普攻默认最多额外弹射 `3` 次，整条链中同一目标最多被命中 `1` 次
  - 弹射命中只造成伤害，不会把自己再当作新一轮普攻重新触发，也不会形成无限循环
  - 弹射是这名英雄最核心的群体压血来源；单点压制不是她的主要职责
- 检测条件：
  - 当前存在合法敌方目标
  - 目标进入攻击范围
  - 普攻间隔转好
- 当前关键参数：
  - 主目标普攻伤害倍率：基于攻击力 `0.85`
  - 弹射次数上限：`3`
  - 每次弹射搜索半径：`3.2`
  - 单次弹射伤害倍率：基于攻击力 `0.48`
  - 攻击距离：`8.2`
  - 普攻间隔：`0.95 秒`
  - 主投射物速度：`15`
  - 弹射链规则：同一次普攻链中的唯一目标集合；不存在合法新目标时立即结束

## 小技能：回旋飞轮

- 技能类型：固定方向往返路径伤害
- 释放目标：优先锁定当前攻击目标所在方向；若当前目标不适合作为方向锚点，则改为选择施法距离内敌人最密集的一名敌方英雄作为路径方向参考
- 效果：
  - 朝锁定方向掷出一只回旋飞轮，沿固定直线外放到最远点，再沿原路返回施法起点
  - 外放和回收都会对路径上的敌方英雄造成伤害
  - 同一目标在外放阶段最多命中 `1` 次，在回收阶段最多命中 `1` 次；若站位合适则允许同一目标吃到两段
  - 该技能的核心价值是“切穿一条局部战线并在返程补压血或收尾”，不是稳定单点爆发按钮
- 检测条件：
  - 小技能 CD 转好
  - 施法距离 `9.2` 内存在合法方向锚点
- 当前关键参数：
  - 锚点施法距离：`9.2`
  - 外放最大距离：`10.5`
  - 路径宽度：`1.5`
  - 外放阶段持续时间：`0.4 秒`
  - 最远点停顿：`0.1 秒`
  - 回收阶段持续时间：`0.4 秒`
  - 外放伤害：基于攻击力 `1.6`
  - 回收伤害：基于攻击力 `1.2`
  - 冷却时间：`6 秒`

## 大招：旋轮风暴

- 技能类型：跟随施法者的持续范围伤害
- 释放目标：自己；优先在敌方前排或突进英雄已经贴近自己时开启，也允许在短兵相接的局部团战中主动压进释放
- 效果：
  - 以自身为中心召唤数列高速旋转的飞轮，形成持续数秒的近身切割区
  - 大招持续期间，进入自己周围范围内的敌方英雄会周期性受到伤害
  - 飞轮只承担“近身持续切割”职责，不附带弹射，不额外复制普攻链，也不附带眩晕、击飞或位移
  - 这招的核心收益是惩罚敌方贴脸、抱团和前压，让短射程射手在被逼近时仍能打出高额范围回击
  - 若标准脆皮完整吃满，应稳定被击杀；中坚应被压进危险线；坦克以明显掉血但仍能存活为主
- 检测条件：
  - 本局尚未释放过大招
  - 优先要求自身 `5` 范围内至少存在 `2` 名敌方英雄
  - 若开战 `45 秒` 后仍未满足理想窗口，则只要自身 `5` 范围内仍存在至少 `1` 名合法敌方目标即可释放
- 当前关键参数：
  - 持续时间：`4.8 秒`
  - 作用半径：`5`
  - 伤害间隔：`0.4 秒`
  - 单跳伤害：基于攻击力 `1.2`
  - 表现飞轮列数：`4`
- 当前关键约束：
  - 主动释放型大招本局仅能释放 `1 次`

## 落地约束

- 轮子妈当前定位是“第四个射手扩展原型”，重点不是再做一个更长射程的后排炮台，而是验证“短射程换群体摊伤”的另一条射手分支
- 普攻弹射必须优先沉淀为共享的 `BasicAttackData` 正式能力，不要把“命中后再找附近敌人”写成 `marksman_004_boomeranger` 的单英雄私有脚本
- 同一次普攻链的主段与每次弹射，都应通过显式的表现标记、跳数或等价事件参数暴露给表现层，让后续特效能稳定接入“主飞轮投射物 -> 弹射拖尾 -> 弹射命中火花”；不要让 `BattleView` 靠 `heroId` 猜测是否播放弹射表现
- `回旋飞轮` 应优先落成共享的“固定方向往返路径命中能力”，默认以施法起点快照作为回收终点；如果未来要改成实时追踪施法者位置，必须先补文档
- `回旋飞轮` 的外放与回收应提供显式的阶段标记或等价表现入口，方便后续把外放拖尾、回收拖尾和两段命中火花分开接，不要等到做特效时才反向改逻辑层
- `旋轮风暴` 的逻辑层应优先复用 `followCaster = true` 的持续区域能力；真正“几列飞轮绕着角色旋转”的观感应落在独立表现控制器或技能专属 `SkillAreaPresentationController` 上，不要把每列飞轮都做成单独物理碰撞体来决定伤害
- 大招的飞轮列数、旋转速度、相位偏移和拖尾节奏都属于表现层参数；伤害仍以正式的持续时间、作用半径和伤害间隔为准，不能让表现层另存一套假半径或假 tick
- 事件与日志至少要能明确记录：单次普攻的弹射命中数、小技能外放命中数、小技能回收命中数，以及大招每跳脉冲的命中人数，方便后续调数和看特效是否跟逻辑一致

## 设计边界

- 轮子妈的核心职责是中短射程范围压血和反贴脸回击，不承担超长射程安全输出职责
- 不应把轮子妈做成高机动突进射手、远程刺客或带强保命无敌的旋风角色
- 不应把轮子妈做成强控制核心；她可以靠群体伤害压血，但不应依赖眩晕、击飞或大范围减速主导团战
- 不应把普攻弹射扩成跨半场自动清全队的全屏连锁；她的群伤必须明显依赖“敌方相互靠近”这一站位条件
- 不应把 `旋轮风暴` 扩成“开大后自己不可选中 / 无敌 / 带位移”的全能保命技；当前它只负责高风险高回报的近身持续伤害
- 第一阶段下，轮子妈优先验证“短射程弹射普攻 + 往返飞轮路径技 + 自身周围旋轮持续大招”这套 AOE 射手模板
